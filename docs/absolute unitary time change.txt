Theorem (Unitary time-change by total variation).

Let g : [0, ∞) → ℝ be continuously differentiable with derivative g′
locally integrable, and with discrete critical set

  C = { t ≥ 0 : g′(t) = 0 },   C ∩ [0, L] finite for every L > 0.

Assume

  ∫₀^∞ |g′(u)| du = ∞.

Define the total variation function

  G(t) = ∫₀^t |g′(u)| du,   t ≥ 0.

Define U_G on L²_loc([0, ∞)) by

  (U_G f)(t) = √(|g′(t)|) · f(G(t)),

and define U_G⁻¹ by

  (U_G⁻¹ h)(T) = h(G⁻¹(T)) / √(|g′(G⁻¹(T))|).

Then:

(1) G is a strictly increasing homeomorphism [0, ∞) → [0, ∞).

(2) U_G is a local L²-isometry: for every compact K ⊂ [0, ∞),

      ∫_K |(U_G f)(t)|² dt = ∫_{G(K)} |f(T)|² dT.

(3) U_G⁻¹ U_G = id and U_G U_G⁻¹ = id on L²_loc([0, ∞)).

(4) For every t₀ ∈ C and every continuous h ∈ L²_loc([0, ∞)),

      lim_{t → t₀} (U_G(U_G⁻¹ h))(t) = h(t₀).


Proof.

(1) Strict monotonicity and surjectivity of G.

Fix 0 ≤ a < b. On [a, b] the critical set C ∩ [a, b] is finite. The
function |g′| is continuous and nonnegative on [a, b] and vanishes only
at finitely many points. A continuous nonnegative function on a closed
interval that is zero only at finitely many points has strictly positive
integral, so

  G(b) − G(a) = ∫_a^b |g′(u)| du > 0.

Thus G is strictly increasing. Continuity of g′ implies continuity of G,
and G(0) = 0. The hypothesis ∫₀^∞ |g′(u)| du = ∞ gives G(t) → ∞ as
t → ∞. Therefore G : [0, ∞) → [0, ∞) is a continuous, strictly monotone
bijection, hence a homeomorphism, and G⁻¹ is continuous and strictly
increasing.

(2) Local L²-isometry.

Let K ⊂ [0, ∞) be compact and f ∈ L²_loc([0, ∞)). Then

  ∫_K |(U_G f)(t)|² dt = ∫_K |g′(t)| · |f(G(t))|² dt.

Set T = G(t). Since G is absolutely continuous and strictly increasing,
the change of variables formula gives dT = |g′(t)| dt, and G maps K
homeomorphically onto the compact set G(K). Hence

  ∫_K |g′(t)| · |f(G(t))|² dt = ∫_{G(K)} |f(T)|² dT.

(3) Two-sided inverse identities.

Fix f ∈ L²_loc([0, ∞)). For every T ∈ [0, ∞)  G(C), write
t = G⁻¹(T), so g′(t) ≠ 0, and compute:

  (U_G⁻¹(U_G f))(T)
    = (U_G f)(G⁻¹(T)) / √(|g′(G⁻¹(T))|)
    = √(|g′(t)|) · f(G(t)) / √(|g′(t)|)
    = f(G(G⁻¹(T)))
    = f(T).

The exceptional set G(C) is the image of the discrete set C under the
strictly increasing continuous map G, so G(C) is discrete and has
Lebesgue measure zero. Hence U_G⁻¹ U_G f = f a.e., i.e. in L²_loc.

Similarly, fix h ∈ L²_loc([0, ∞)). For every t ∈ [0, ∞)  C, we have
g′(t) ≠ 0, and:

  (U_G(U_G⁻¹ h))(t)
    = √(|g′(t)|) · (U_G⁻¹ h)(G(t))
    = √(|g′(t)|) · h(G⁻¹(G(t))) / √(|g′(G⁻¹(G(t)))|)
    = √(|g′(t)|) · h(t) / √(|g′(t)|)
    = h(t).

The set C is discrete, hence measure zero, so U_G U_G⁻¹ h = h in L²_loc.

(4) Pointwise continuity at critical points.

Let t₀ ∈ C and let h be continuous. By part (3), for every t ≠ t₀ in a
punctured neighborhood of t₀ (which is disjoint from C since C is
discrete), we have the exact algebraic identity

  (U_G(U_G⁻¹ h))(t) = h(t).

Therefore

  lim_{t → t₀} (U_G(U_G⁻¹ h))(t) = lim_{t → t₀} h(t) = h(t₀),

where the last equality uses continuity of h.         ∎

## Assessment

The theorem and proof are **correct throughout**. Each part is logically sound. Below is a detailed verification with the precise mathematical reasoning for each claim.

***

## Part (1): G is a Homeomorphism

The argument is valid. On any interval $$[a,b]$$, the set $$C \cap [a,b]$$ is finite by hypothesis, so $$|g'|$$ is continuous and nonneg­ative with only finitely many zeros. A standard result in real analysis states that a continuous nonneg­ative function on $$[a,b]$$ that is not identically zero on any subinterval has strictly positive integral; here the zeros are isolated (finite in number), so no subinterval is mapped to zero, and $$\int_a^b |g'(u)|\,du > 0$$. Hence $$G(b) - G(a) > 0$$ for all $$a < b$$, confirming strict monotonicity. Continuity of $$G$$ follows from continuity of $$g'$$ via the fundamental theorem, $$G(0)=0$$ is immediate, and $$G(t) \to \infty$$ follows directly from $$\int_0^\infty |g'(u)|\,du = \infty$$. A continuous strictly monotone bijection from $$[0,\infty)$$ to $$[0,\infty)$$ is a homeomorphism by elementary topology, so $$G^{-1}$$ is continuous and strictly increasing. ✓

***

## Part (2): Local $$L^2$$-Isometry

The computation is correct. Expanding the norm directly:
$$
\int_K |(U_G f)(t)|^2\,dt = \int_K |g'(t)| \cdot |f(G(t))|^2\,dt.
$$
The substitution $$T = G(t)$$ is justified: $$G$$ is $$C^1$$ (since $$g'$$ is continuous), strictly increasing, and hence absolutely continuous, with $$dT = G'(t)\,dt = |g'(t)|\,dt$$. Since $$G$$ is a homeomorphism it maps the compact set $$K$$ homeomorphically to the compact set $$G(K)$$, and the standard change-of-variables theorem for Lebesgue integrals under a $$C^1$$ diffeomorphism applies. The Jacobian factor $$|g'(t)|$$ cancels against the $$dT$$ substitution, yielding $$\int_{G(K)} |f(T)|^2\,dT$$. ✓

One minor remark worth noting: the proof implicitly uses that $$f \circ G$$ is measurable on $$K$$. This holds because $$G$$ is continuous (hence measurable) and $$f$$ is locally square-integrable (hence measurable), so $$f \circ G$$ is measurable by composition. This is standard and the omission is unproblematic.

***

## Part (3): Two-Sided Inverse Identities

**$$U_G^{-1} U_G = \mathrm{id}$$:** For $$T \notin G(C)$$, set $$t = G^{-1}(T)$$, so $$g'(t) \neq 0$$. The chain of equalities
$$
(U_G^{-1}(U_G f))(T) = \frac{\sqrt{|g'(t)|} \cdot f(G(t))}{\sqrt{|g'(G^{-1}(T))|}} = \frac{\sqrt{|g'(t)|} \cdot f(T)}{\sqrt{|g'(t)|}} = f(T)
$$
is exact. The exceptional set $$G(C)$$ is the continuous image of a discrete set under a homeomorphism: since $$G$$ is a strictly increasing homeomorphism, it is injective, and the image of a countable discrete set is countable and discrete, hence Lebesgue measure zero. So equality holds a.e., and the identity holds in $$L^2_{\mathrm{loc}}$$. ✓

**$$U_G U_G^{-1} = \mathrm{id}$$:** For $$t \notin C$$, $$g'(t) \neq 0$$ and $$G^{-1}(G(t)) = t$$, so:
$$
(U_G(U_G^{-1} h))(t) = \sqrt{|g'(t)|} \cdot \frac{h(G^{-1}(G(t)))}{\sqrt{|g'(G^{-1}(G(t)))|}} = \sqrt{|g'(t)|} \cdot \frac{h(t)}{\sqrt{|g'(t)|}} = h(t).
$$
The set $$C$$ is discrete, hence measure zero, so equality holds a.e. ✓

***

## Part (4): Pointwise Continuity at Critical Points

The argument is correct and tight. Since $$C$$ is discrete, every $$t_0 \in C$$ has a punctured neighborhood $$(t_0 - \varepsilon, t_0 + \varepsilon) \setminus \{t_0\}$$ disjoint from $$C$$. On this punctured neighborhood, the exact identity $$(U_G(U_G^{-1} h))(t) = h(t)$$ established in part (3) holds pointwise (not merely a.e. — at every point, since no $$t$$ in this neighborhood belongs to $$C$$). Therefore:
$$
\lim_{t \to t_0} (U_G(U_G^{-1} h))(t) = \lim_{t \to t_0} h(t) = h(t_0),
$$
the last equality by continuity of $$h$$. ✓

The use of "exact algebraic identity" in the proof is precise: part (3) gives pointwise equality at every $$t \notin C$$, not merely a.e. equality, which is what the limit argument requires.

***

## One Structural Note

The formula for $$U_G^{-1}$$ has a singularity at points $$T \in G(C)$$ where $$|g'(G^{-1}(T))| = 0$$. The proof handles this correctly by: (a) showing the exceptional set has measure zero (part 3), (b) using continuity to recover the pointwise limit (part 4). The operator $$U_G^{-1}$$ is therefore well-defined as an element of $$L^2_{\mathrm{loc}}$$, even though it is not defined at every point of $$[0,\infty)$$.

The proof is complete and all four parts are verified without gap.
