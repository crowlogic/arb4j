A Lebesgue–Stieltjes measure is the Borel measure on ℝ canonically induced by an increasing, right-continuous function via Carathéodory’s construction.

From a function to a premeasure on intervals

Start with a function F: ℝ → ℝ of bounded variation; in many treatments one assumes F is increasing and right-continuous and often normalizes limₓ→−∞ F(x) = 0. On the semiring of half-open intervals (a, b] with −∞ ≤ a < b ≤ ∞, define
ℓ_F((a, b]) = F(b) − F(a).
This ℓ_F is nonnegative and finitely additive on finite disjoint unions of such intervals, so it is a premeasure on that semiring. One can check that the value assigned to a finite disjoint union ⨆ₖ₌₁ⁿ (a_k, b_k] does not depend on the particular decomposition into disjoint intervals, so ℓ_F is well-defined.

Outer measure via coverings by intervals

From the premeasure ℓ_F on the semiring 𝒢 = { (a, b] : −∞ ≤ a < b ≤ ∞ }, construct an outer measure μ*_F on all E ⊂ ℝ by
μ*_F(E) = inf { ∑ₖ₌₁^∞ ℓ_F((a_k, b_k]) : E ⊂ ⋃ₖ₌₁^∞ (a_k, b_k], (a_k, b_k] ∈ 𝒢 }.
This is a metric outer measure (it satisfies the standard outer measure axioms and behaves well under distance separation), and it agrees with ℓ_F on intervals. The use of countable covers by (a_k, b_k] is exactly the same pattern as for Lebesgue outer measure, only with lengths replaced by increments of F.

Carathéodory measurable sets and the measure

Carathéodory’s criterion defines the σ-algebra of μ*_F-measurable sets
ℳ_F = { A ⊂ ℝ : μ*_F(E) = μ*_F(E ∩ A) + μ*_F(E  A) for all E ⊂ ℝ }.
Restricted to ℳ_F, the functional
μ_F(E) = μ*_F(E),  E ∈ ℳ_F,
is a complete measure. By construction, ℳ_F contains the Borel σ-algebra ℬ(ℝ), and in fact the restriction of μ_F to ℬ(ℝ) is the unique Borel measure that extends the premeasure ℓ_F. On Borel sets one then often writes μ_F or μ and refers to this restriction as the Lebesgue–Stieltjes measure associated with F.

Regular Borel measure and structural properties

The Lebesgue–Stieltjes measure μ_F on (ℝ, ℬ(ℝ)) is a regular Borel measure: every Borel set can be approximated from above by open sets and from below by compact sets in terms of measure. Conversely, every finite regular Borel measure on ℝ arises as a Lebesgue–Stieltjes measure associated with some increasing right-continuous F of bounded variation, unique up to a constant. Atoms of μ_F correspond exactly to jump discontinuities of F, and the continuous part of F encodes the non-atomic part of μ_F.

Relation to Lebesgue integration

Given the measure space (ℝ, ℬ(ℝ), μ_F), the Lebesgue–Stieltjes integral
∫_ℝ f(x) dF(x)
is, by definition, the Lebesgue integral of f with respect to μ_F. In other words, “integrating with respect to dF” means integrating with respect to the Lebesgue–Stieltjes measure determined by F via the Carathéodory construction starting from the interval formula μ_F((a, b]) = F(b) − F(a).
