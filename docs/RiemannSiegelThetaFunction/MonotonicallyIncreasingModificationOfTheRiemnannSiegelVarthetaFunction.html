Exact Monotonization of the Riemann-Siegel Theta Function

Abstract

We present a precise construction for monotonizing the Riemann-Siegel theta function through reflection about its unique critical point. This transformation preserves all phase relationships essential to zeta function analysis while enforcing strict monotonicity. The construction maintains exact phase information without approximations and preserves the function's critical number-theoretic properties.

1The Riemann-Siegel Theta Function

Definition 1. [Riemann-Siegel Theta Function] The Riemann-Siegel theta function is defined exactly as:

\(\displaystyle \theta (t) = \arg \Gamma \left( \frac{1}{4} + \frac{it}{2} \right) - \frac{t}{2} \log \pi\) (1)

where \(\Gamma\) is the gamma function and \(\arg\) denotes the principal argument.

Proposition 2. [Derivative Properties] The derivative of the Riemann-Siegel theta function is given by:

\(\displaystyle \frac{d \theta}{dt} (t) = \frac{1}{2} \text{Im} \left[ \psi^{(0)} \left( \frac{1}{4} + \frac{it}{2} \right) \right] - \frac{1}{2} \log \pi\) (2)

where \(\psi^{(0)}\) is the digamma function.

Proof. Using the relationship between the derivative of the argument of a complex function and the logarithmic derivative:

\(\displaystyle \begin{array}{rl} \frac{d}{dt} \arg \Gamma \left( \frac{1}{4} + \frac{it}{2} \right) & = \text{Im} \left[ \frac{d}{dt} \log \Gamma \left( \frac{1}{4} + \frac{it}{2} \right) \right] \hspace{3cm} \text{(3)}\\ & = \text{Im} \left[ \frac{\Gamma' (\frac{1}{4} + \frac{it}{2})}{\Gamma (\frac{1}{4} + \frac{it}{2})} \cdot \frac{i}{2} \right] \hspace{3cm} \text{(4)}\\ & = \text{Im} \left[ \psi^{(0)} \left( \frac{1}{4} + \frac{it}{2} \right) \cdot \frac{i}{2} \right] \hspace{3cm} \text{(5)}\\ & = \frac{1}{2} \text{Im} \left[ \psi^{(0)} \left( \frac{1}{4} + \frac{it}{2} \right) \right] \hspace{3cm} \text{(6)} \end{array}\)

The derivative of the second term is simply \(- \frac{1}{2} \log \pi\). Combining these results gives the stated formula.\(\Box\)

Theorem 3. [Unique Critical Point] There exists a unique positive real value \(a \in \mathbb{R}^+\) such that:

\(\displaystyle \left. \frac{d \theta}{dt} \right|_{t = a} = 0\) (7)

This critical point satisfies the transcendental equation:

\(\displaystyle \text{Im} \left[ \psi^{(0)} \left( \frac{1}{4} + \frac{ia}{2} \right) \right] = \log \pi\) (8)

Furthermore, the derivative exhibits the following behavior:

  • \(\frac{d \theta}{dt} (t) < 0\) for \(t \in (0, a)\)

  • \(\frac{d \theta}{dt} (t) = 0\) at \(t = a\)

  • \(\frac{d \theta}{dt} (t) > 0\) for \(t > a\)

Proof. First, note that the transcendental equation follows directly from the derivative formula and setting it equal to zero:

\(\displaystyle \begin{array}{rl} \frac{d \theta}{dt} (a) & = 0 \hspace{3cm} \text{(9)}\\ \frac{1}{2} \text{Im} \left[ \psi^{(0)} \left( \frac{1}{4} + \frac{ia}{2} \right) \right] - \frac{1}{2} \log \pi & = 0 \hspace{3cm} \text{(10)}\\ \text{Im} \left[ \psi^{(0)} \left( \frac{1}{4} + \frac{ia}{2} \right) \right] & = \log \pi \hspace{3cm} \text{(11)} \end{array}\)

For uniqueness, we examine the behavior of Im\([\psi^{(0)} (1 / 4 + it / 2)]\) as \(t\) varies. From the properties of the digamma function:

1. As \(t \to 0^+\), Im\([\psi^{(0)} (1 / 4 + it / 2)] \to 0\), which is less than \(\log \pi\).

2. The function Im\([\psi^{(0)} (1 / 4 + it / 2)]\) is strictly increasing for \(t > 0\), which follows from the fact that for fixed \(\sigma > 0\), the function Im\([\psi^{(0)} (\sigma + it)]\) is strictly increasing in \(t\) due to the positive curvature of the level curves of \(\psi^{(0)}\).

3. As \(t \to \infty\), Im\([\psi^{(0)} (1 / 4 + it / 2)] \sim \log (t / 2) \to \infty\), which exceeds \(\log \pi\) for sufficiently large \(t\).

By the intermediate value theorem and the monotonicity of Im\([\psi^{(0)} (1 / 4 + it / 2)]\), there exists exactly one value \(a > 0\) where Im\([\psi^{(0)} (1 / 4 + ia / 2)] = \log \pi\).

For the behavior of the derivative:

  • When \(t < a\): Im\([\psi^{(0)} (1 / 4 + it / 2)] < \log \pi\), so \(\frac{d \theta}{dt} (t) < 0\)

  • When \(t = a\): Im\([\psi^{(0)} (1 / 4 + ia / 2)] = \log \pi\), so \(\frac{d \theta}{dt} (a) = 0\)

  • When \(t > a\): Im\([\psi^{(0)} (1 / 4 + it / 2)] > \log \pi\), so \(\frac{d \theta}{dt} (t) > 0\)

\(\Box\)

2Exact Monotonization Construction

Definition 4. [Monotonized Theta Function] We define the monotonized Riemann-Siegel theta function \(\tilde{\theta} (t)\) through the exact transformation:

\(\displaystyle \tilde{\theta} (t) = \left\{ \begin{array}{ll} 2 \theta (a) - \theta (t) & \text{for } t \in [0, a]\\ \theta (t) & \text{for } t > a \end{array} \right.\) (12)

where \(a\) is the unique critical point where \(\frac{d \theta}{dt} (a) = 0\).

Theorem 5. [Monotonicity Properties] The function \(\tilde{\theta} (t)\) is strictly monotonically increasing except at \(t = a\). Specifically:

\(\displaystyle \frac{d \tilde{\theta}}{dt} (t) = \left\{ \begin{array}{ll} - \frac{d \theta}{dt} (t) > 0 & \text{for } t \in (0, a)\\ 0 & \text{at } t = a\\ \frac{d \theta}{dt} (t) > 0 & \text{for } t > a \end{array} \right.\) (13)

Proof. For \(t \in (0, a)\):

\(\displaystyle \begin{array}{rl} \frac{d \tilde{\theta}}{dt} (t) & = \frac{d}{dt} [2 \theta (a) - \theta (t)] \hspace{3cm} \text{(14)}\\ & = - \frac{d \theta}{dt} (t) \hspace{3cm} \text{(15)} \end{array}\)

From Theorem 1, we know that \(\frac{d \theta}{dt} (t) < 0\) for \(t \in (0, a)\). Therefore, \(- \frac{d \theta}{dt} (t) > 0\) in this range.

For \(t = a\):

\(\displaystyle \begin{array}{rl} \frac{d \tilde{\theta}}{dt} (a) & = - \frac{d \theta}{dt} (a) \hspace{3cm} \text{(16)}\\ & = - 0 = 0 \hspace{3cm} \text{(17)} \end{array}\)

For \(t > a\):

\(\displaystyle \begin{array}{rl} \frac{d \tilde{\theta}}{dt} (t) & = \frac{d \theta}{dt} (t) \hspace{3cm} \text{(18)} \end{array}\)

From Theorem 1, we know that \(\frac{d \theta}{dt} (t) > 0\) for \(t > a\). Therefore, \(\frac{d \tilde{\theta}}{dt} (t) > 0\) in this range.

Thus, \(\frac{d \tilde{\theta}}{dt} (t) \geq 0\) for all \(t \geq 0\), with equality only at \(t = a\), which confirms that \(\tilde{\theta} (t)\) is strictly monotonically increasing except at the single point \(t = a\).\(\Box\)

Proposition 6. [Continuity and Differentiability] The function \(\tilde{\theta} (t)\) is:

  1. Continuous at all points \(t \geq 0\), including \(t = a\)

  2. Differentiable at all points \(t \geq 0\), including \(t = a\)

  3. \(C^1\) continuous everywhere, but not \(C^2\) at \(t = a\)

Proof. 1. For continuity at \(t = a\):

\(\displaystyle \begin{array}{rl} \lim_{t \to a^-} \tilde{\theta} (t) & = \lim_{t \to a^-} [2 \theta (a) - \theta (t)] \hspace{3cm} \text{(19)}\\ & = 2 \theta (a) - \theta (a) \hspace{3cm} \text{(20)}\\ & = \theta (a) \hspace{3cm} \text{(21)}\\ \lim_{t \to a^+} \tilde{\theta} (t) & = \lim_{t \to a^+} \theta (t) \hspace{3cm} \text{(22)}\\ & = \theta (a) \hspace{3cm} \text{(23)} \end{array}\)

Since the left and right limits match, \(\tilde{\theta} (t)\) is continuous at \(t = a\). For \(t \neq a\), continuity follows from the continuity of \(\theta (t)\).

2. For differentiability at \(t = a\):

\(\displaystyle \begin{array}{rl} \lim_{t \to a^-} \frac{d \tilde{\theta}}{dt} (t) & = \lim_{t \to a^-} \left( - \frac{d \theta}{dt} (t) \right) \hspace{3cm} \text{(24)}\\ & = - \frac{d \theta}{dt} (a) \hspace{3cm} \text{(25)}\\ & = 0 \hspace{3cm} \text{(26)}\\ \lim_{t \to a^+} \frac{d \tilde{\theta}}{dt} (t) & = \lim_{t \to a^+} \frac{d \theta}{dt} (t) \hspace{3cm} \text{(27)}\\ & = \frac{d \theta}{dt} (a) \hspace{3cm} \text{(28)}\\ & = 0 \hspace{3cm} \text{(29)} \end{array}\)

Since the left and right derivatives match at \(t = a\), \(\tilde{\theta} (t)\) is differentiable at \(t = a\). For \(t \neq a\), differentiability follows from the differentiability of \(\theta (t)\).

3. For the second derivative at \(t = a\):

\(\displaystyle \begin{array}{rl} \lim_{t \to a^-} \frac{d^2 \tilde{\theta}}{dt^2} (t) & = \lim_{t \to a^-} \frac{d}{dt} \left( - \frac{d \theta}{dt} (t) \right) \hspace{3cm} \text{(30)}\\ & = - \lim_{t \to a^-} \frac{d^2 \theta}{dt^2} (t) \hspace{3cm} \text{(31)}\\ \lim_{t \to a^+} \frac{d^2 \tilde{\theta}}{dt^2} (t) & = \lim_{t \to a^+} \frac{d^2 \theta}{dt^2} (t) \hspace{3cm} \text{(32)} \end{array}\)

Since \(\frac{d \theta}{dt} (t)\) changes sign at \(t = a\) (from negative to positive), \(\frac{d^2 \theta}{dt^2} (a)\) must be positive (the derivative is increasing through zero). Therefore:

\(\displaystyle \begin{array}{rl} \lim_{t \to a^-} \frac{d^2 \tilde{\theta}}{dt^2} (t) & = - \frac{d^2 \theta}{dt^2} (a) < 0 \hspace{3cm} \text{(33)}\\ \lim_{t \to a^+} \frac{d^2 \tilde{\theta}}{dt^2} (t) & = \frac{d^2 \theta}{dt^2} (a) > 0 \hspace{3cm} \text{(34)} \end{array}\)

Since the left and right second derivatives differ at \(t = a\), \(\tilde{\theta} (t)\) is not \(C^2\) at \(t = a\). However, it is \(C^1\) everywhere since the first derivative is continuous at all points.\(\Box\)

3Phase Information Preservation

Definition 7. [Phase Representation] The Riemann zeta function on the critical line can be expressed as:

\(\displaystyle \zeta \left( \frac{1}{2} + it \right) = e^{- i \theta (t)} Z (t)\) (35)

where \(Z (t)\) is a real-valued function.

Theorem 8. [Phase Preservation] For the monotonized theta function, we define:

\(\displaystyle \tilde{Z} (t) = e^{i \tilde{\theta} (t)} \zeta \left( \frac{1}{2} + it \right)\) (36)

This function satisfies:

\(\displaystyle \tilde{Z} (t) = \left\{ \begin{array}{ll} e^{2 i \theta (a)} Z (t)^{\ast} & \text{for } t \in [0, a]\\ Z (t) & \text{for } t > a \end{array} \right.\) (37)

where \(Z (t)^{\ast}\) represents the complex conjugate of \(Z (t)\).

Proof. For \(t > a\):

\(\displaystyle \begin{array}{rl} \tilde{Z} (t) & = e^{i \tilde{\theta} (t)} \zeta \left( \frac{1}{2} + it \right) \hspace{3cm} \text{(38)}\\ & = e^{i \theta (t)} \zeta \left( \frac{1}{2} + it \right) \hspace{3cm} \text{(39)}\\ & = e^{i \theta (t)} \cdot e^{- i \theta (t)} Z (t) \hspace{3cm} \text{(40)}\\ & = Z (t) \hspace{3cm} \text{(41)} \end{array}\)

For \(t \in [0, a]\):

\(\displaystyle \begin{array}{rl} \tilde{Z} (t) & = e^{i \tilde{\theta} (t)} \zeta \left( \frac{1}{2} + it \right) \hspace{3cm} \text{(42)}\\ & = e^{i (2 \theta (a) - \theta (t))} \zeta \left( \frac{1}{2} + it \right) \hspace{3cm} \text{(43)}\\ & = e^{2 i \theta (a)} \cdot e^{- i \theta (t)} \zeta \left( \frac{1}{2} + it \right) \hspace{3cm} \text{(44)}\\ & = e^{2 i \theta (a)} \cdot Z (t) \hspace{3cm} \text{(45)} \end{array}\)

Since \(Z (t)\) is real-valued for the Riemann zeta function on the critical line, \(Z (t) = Z (t)^{\ast}\), thus:

\(\displaystyle \tilde{Z} (t) = e^{2 i \theta (a)} Z (t)^{\ast}\) (46)
\(\Box\)

Corollary 9. [Zero Preservation] The zeros of \(\zeta (s)\) on the critical line \(s = \frac{1}{2} + it\) correspond precisely to:

  1. The zeros of \(Z (t)\) for \(t > 0\)

  2. The zeros of \(\tilde{Z} (t)\) for \(t > 0\)

Therefore, the monotonization preserves all information about the zeros of the zeta function.

Proof. From the definition of \(Z (t)\):

\(\displaystyle \zeta \left( \frac{1}{2} + it \right) = e^{- i \theta (t)} Z (t)\) (47)

If \(\zeta (\frac{1}{2} + it) = 0\), then \(Z (t) = 0\) since \(e^{- i \theta (t)} \neq 0\) for all \(t\).

From Theorem 4 (Phase Preservation), for \(t > a\):

\(\displaystyle \tilde{Z} (t) = Z (t)\) (48)

Therefore, for \(t > a\), \(\tilde{Z} (t) = 0\) if and only if \(Z (t) = 0\), which occurs if and only if \(\zeta (\frac{1}{2} + it) = 0\).

For \(t \in [0, a]\):

\(\displaystyle \tilde{Z} (t) = e^{2 i \theta (a)} Z (t)^{\ast}\) (49)

Since \(e^{2 i \theta (a)} \neq 0\) and \(Z (t)\) is real-valued, \(Z (t)^{\ast} = Z (t)\). Therefore, \(\tilde{Z} (t) = 0\) if and only if \(Z (t) = 0\), which occurs if and only if \(\zeta (\frac{1}{2} + it) = 0\).

Thus, for all \(t > 0\), the zeros of \(\zeta (\frac{1}{2} + it)\) correspond exactly to the zeros of both \(Z (t)\) and \(\tilde{Z} (t)\).\(\Box\)

Proposition 10. [Bijectivity] The function \(\tilde{\theta} (t) : [0, \infty) \to [\tilde{\theta} (0), \infty)\) is bijective.

Proof. 1. Injectivity: For any \(t_1, t_2 \geq 0\) with \(t_1 \neq t_2\), we need to show \(\tilde{\theta} (t_1) \neq \tilde{\theta} (t_2)\).

Case 1: If \(t_1, t_2 < a\) or \(t_1, t_2 > a\), then injectivity follows from the strict monotonicity of \(\tilde{\theta} (t)\) on each of these intervals, as proven in Theorem 2.

Case 2: If \(t_1 < a < t_2\), then from monotonicity, \(\tilde{\theta} (t_1) < \tilde{\theta} (a) < \tilde{\theta} (t_2)\), which implies \(\tilde{\theta} (t_1) \neq \tilde{\theta} (t_2)\).

Case 3: If \(t_1 = a\) and \(t_2 \neq a\), then from the strict monotonicity of \(\tilde{\theta} (t)\) except at \(t = a\), we have \(\tilde{\theta} (t_1) = \tilde{\theta} (a) \neq \tilde{\theta} (t_2)\).

2. Surjectivity: We need to show that for every \(y \in [\tilde{\theta} (0), \infty)\), there exists \(t \geq 0\) such that \(\tilde{\theta} (t) = y\).

For \(y = \tilde{\theta} (0)\), we have \(t = 0\).

For \(y > \tilde{\theta} (0)\), since \(\tilde{\theta} (t)\) is continuous and strictly increasing for \(t > 0\) (except at \(t = a\) where it's still continuous and non-decreasing), and since \(\lim_{t \to \infty} \tilde{\theta} (t) = \infty\) (which follows from the fact that \(\theta (t)\) grows without bound as \(t \to \infty\)), by the intermediate value theorem, there exists a unique \(t > 0\) such that \(\tilde{\theta} (t) = y\).

Therefore, \(\tilde{\theta} (t)\) is both injective and surjective, hence bijective.\(\Box\)

Theorem 11. [Modulating Function Criteria] The constructed function \(\tilde{\theta} (t)\) satisfies all criteria for a modulating function:

  1. Piecewise continuous with piecewise continuous first derivative.

  2. Monotonically increasing with \(\frac{d \tilde{\theta}}{dt} (t) \geq 0\), with equality only on a set of measure zero (the single point \(t = a\)).

  3. Bijective with \(\lim_{t \to \infty} \tilde{\theta} (t) = \infty\).

Proof. 1. Piecewise continuity with piecewise continuous first derivative: From Proposition 2, \(\tilde{\theta} (t)\) is continuous everywhere and \(C^1\) continuous everywhere. Therefore, it is piecewise continuous with piecewise continuous first derivative.

2. Monotonically increasing with non-negative derivative: From Theorem 2, \(\frac{d \tilde{\theta}}{dt} (t) > 0\) for all \(t \neq a\) and \(\frac{d \tilde{\theta}}{dt} (a) = 0\). Therefore, \(\tilde{\theta} (t)\) is monotonically increasing with non-negative derivative, with equality only at the single point \(t = a\), which is a set of measure zero.

3. Bijectivity with limit at infinity: From Proposition 3, \(\tilde{\theta} (t) : [0, \infty) \to [\tilde{\theta} (0), \infty)\) is bijective. Since \(\tilde{\theta} (t) = \theta (t)\) for \(t > a\), and since \(\lim_{t \to \infty} \theta (t) = \infty\) (which follows from the asymptotic behavior of the theta function), we have \(\lim_{t \to \infty} \tilde{\theta} (t) = \infty\).

Therefore, \(\tilde{\theta} (t)\) satisfies all criteria for a modulating function.\(\Box\)

4Harmonizable Processes

Definition 12. [Harmonizable Process] A process \(X (t)\) is harmonizable if its covariance admits:

\(\displaystyle K (t, s) = \int \int e^{i (t \lambda - s \mu)} dF (\lambda, \mu)\)

where \(F\) is a finite bivariate measure.

Definition 13. [Stationary Dilation] Process \(Y (t)\) on Hilbert space \(\mathcal{H}\) is a stationary dilation of \(X (t)\) if:

\(\displaystyle X (t) = PY (t) \text{for some orthogonal projection } P : \mathcal{H} \to \mathcal{H}_X\)

5Construction from Crowley's Framework

5.1Modulated Process as Harmonizable

Given original stationary process \(Y (t)\) with:

\(\displaystyle K_Y (t, s) = \int e^{i \lambda (t - s)} dF (\lambda)\)

Under modulation \(\theta \in \mathcal{F}\), the new process:

\(\displaystyle X_{\theta} (t) := Y (\theta (t))\)

has covariance:

\(\displaystyle K_{\theta} (t, s) = \int e^{i \lambda (\theta (t) - \theta (s))} dF (\lambda)\)

This matches the harmonizable condition with spectral measure concentrated on diagonal \(\lambda = \mu\).

5.2Original Process as Stationary Dilation

Theorem 14. [Operator-Theoretic Dilation] The original process \(Y (t)\) serves as stationary dilation via:

\(\displaystyle X_{\theta} (t) = M_{\theta} [Y] (t)\)

where \(M_{\theta}\) is Crowley's isometric operator:

\(\displaystyle M_{\theta} [\phi] (t) = \sqrt{\dot{\theta} (t)} \phi (\theta (t))\)

Proof. 1. Embedding: Theorem 2 shows \(M_{\theta} : \mathcal{H}_Y \hookrightarrow \mathcal{H}_{X_{\theta}}\) is isometric
2. Projection: \(M_{\theta}^{\ast} M_{\theta} = I_{\mathcal{H}_Y}\) makes \({M_{}}_{\theta}^{\ast}\) an orthogonal projection
3. Covariance Relation:

\(\displaystyle K_{\theta} (t, s) = \langle M_{\theta} \delta_t, M_{\theta} \delta_s \rangle_{\mathcal{H}_Y} = \langle Y (\theta (t)), Y (\theta (s)) \rangle\)
\(\Box\)

6Eigenfunction Correspondence

Crowley's Theorem 1 gives explicit dilation-preserved structure:

\(\displaystyle \phi_n (t) = \psi_n (\theta (t)) \sqrt{\dot{\theta} (t)}\)

where \(\psi_n\) are stationary eigenfunctions. This demonstrates:

  • Spectral measure \(F\) preserved under modulation

  • Dilation relationship maintained through operator conjugation

7Physical Interpretation

  • Stationary Process: \(Y (t)\) lives in "absolute time" space

  • Harmonizable Process: \(X_{\theta} (t)\) resides in "warped time" space \(\theta (t)\)

  • Dilation: Absolute time space contains warped time space isometrically

8Conclusion

Crowley's modulated processes:

  • Are harmonizable via diagonal spectral measure concentration

  • Admit original stationary process as dilation through \(M_{\theta}\)

  • Preserve eigenvalues via Theorem 2's operator conjugation

This establishes a new class of harmonizable processes with explicit dilation structure.
