Theorem (Radon–Nikodym derivative of γ under the shift operator).

Let γ be the Borel measure on (0,∞) with density dγ(t) ≔ ψ′(t) dt, where

    ψ′(t) ≔ θ′(t) ∣ζ(½+it)∣²,

and θ(t) = Im log Γ(¼ + ½it) − ½t log π is the Riemann–Siegel phase.
The function ψ is C¹ on (0,∞) and ψ′ is continuous. Let T_h be the shift
operator T_h(t) = t + h. Then the Radon–Nikodym derivative of T_h γ with
respect to γ is

    d(T_h γ) / dγ (t)  =  ψ′(t−h) / ψ′(t)  =  θ′(t−h)∣ζ(½+i(t−h))∣² / θ′(t)∣ζ(½+it)∣²

for all t ∈ (0,∞) with ψ′(t) ≠ 0.

Proof.

Both γ and T_h γ are absolutely continuous with respect to Lebesgue
measure λ, with continuous densities dγ/dλ = ψ′(t) and d(T_h γ)/dλ =
ψ′(t−h). The chain rule for Radon–Nikodym derivatives gives

    d(T_h γ) / dγ  =  [d(T_h γ) / dλ] / [dγ / dλ]  =  ψ′(t−h) / ψ′(t)

at every t where ψ′(t) ≠ 0. Since θ′(t) > 0 for t > 0, the zeros of ψ′
are exactly the zeros of ζ(½+it). The function ζ is holomorphic and not
identically zero, so its zeros are isolated. A set of isolated points in ℝ
is countable, hence λ-null, hence γ-null. The same holds for the zeros of
ψ′(t−h). Therefore γ and T_h γ are equivalent modulo null sets, and the
Radon–Nikodym derivative is the ratio stated above.  ∎

Remark. The appearance of ∣ζ∣² in both numerator and denominator is the
mechanism: the zeros of ψ′(t−h) and ψ′(t) are both isolated and countable,
so every null set of one measure is a null set of the other.
