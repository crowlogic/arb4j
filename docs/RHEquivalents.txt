RH EQUIVALENCES: POSITIVITY, SPECTRAL THEORY, AND THE ISOMETRIC SECTION ANALOGY
═══════════════════════════════════════════════════════════════════════════════════


§1  ARITHMETIC EQUIVALENCES
────────────────────────────────────────────────────────────────────────────────

The Mertens function M(x) = Σ_{n ≤ x} μ(n) satisfies M(x) = O(x^{1/2+ε}) for
all ε > 0 if and only if RH holds. Robin's inequality σ(n) < e^γ n log log n for
all n > 5040, Lagarias's harmonic number reformulation, and Nicolas's criterion on
Euler's totient at primorial values are each equivalent to RH. The Franel–Landau
theorem identifies RH with the condition that the discrepancy of Farey fractions
from uniform spacing on [0,1] is O(n^{1/2+ε}), and the Redheffer matrix
determinant equals M(n), giving a linear-algebraic restatement.


§2  ANALYTIC EQUIVALENCES
────────────────────────────────────────────────────────────────────────────────

The Riesz criterion, Hardy–Littlewood criterion, Speiser's result that ζ′(s) has
no zeros in 0 < ℜ(s) < 1/2 iff RH holds, and Li's criterion — positivity of all

    λₙ = Σ_ρ [ 1 − (1 − 1/ρ)ⁿ ],   n ≥ 1

— are analytic equivalences. The de Bruijn–Newman constant Λ ≤ 0 is equivalent to
RH, with Rodgers–Tao proving Λ ≥ 0, so RH is exactly Λ = 0.

The Nyman–Beurling criterion states that RH is equivalent to the characteristic
function 𝟏_{(0,1)} lying in the L²(0,1) closure of the span of the dilates
{ ρ(θ/n) : n ∈ ℕ, θ ∈ (0,1] }, where ρ denotes fractional part. The
Baez-Duarte strengthening replaces the full span by the subspace generated by
{ ρ(1/n) : n ∈ ℕ } alone. In probabilistic extensions this closure problem can
be reformulated with random dilation factors; equivalences with RH persist for
two main classes (exponential and concentrated-around-1 distributions).

Weil's criterion (1952, extended 1972) is the following precise statement: RH for
all Dirichlet L-functions is equivalent to the negativity of the distribution

    W(g) = ĝ(0) + ĝ(1) − Σ_ρ ĝ(ρ)

as a functional on the convolution algebra C^∞_c(ℝˣ₊), restricted to functions
vanishing on all zeta zeros. Equivalently, Bombieri showed that the associated
quadratic functional — obtained from the explicit formula — is positive
semidefinite if and only if RH is true, and positive definite on functions
supported in a sufficiently small interval regardless. If RH fails with exactly k
zeros off the critical line, the number of negative eigenvalues of finite
truncations of this quadratic form equals k/2.


§3  SPECTRAL AND OPERATOR-THEORETIC EQUIVALENCES
────────────────────────────────────────────────────────────────────────────────

The Hilbert–Pólya conjecture posits a self-adjoint operator whose eigenvalues are
the imaginary parts of the nontrivial zeta zeros; no such operator has been
explicitly constructed. Berry and Keating proposed that the classical Hamiltonian
H = xp on the half-line, with a phase-space cutoff |x| ≥ ℓ_x, |p| ≥ ℓ_p, yields
a semiclassical spectrum reproducing the average density of Riemann zeros. At the
quantum level, normal-order quantization of xp does not exhibit the exact Riemann
zeros in its spectrum; realizing them requires additional boundary interaction
terms or non-local operators, and the model is not unique. In a refined version,
adding interaction terms depending on two boundary wave functions gives a continuum
spectrum with discrete embedded bound states whose resonances converge to the
Riemann zeros, exploiting a duality symmetry related to the functional equation.

Montgomery's pair correlation conjecture connects the statistics of spacings
between imaginary parts of zeros to the GUE (Gaussian Unitary Ensemble)
correlation functions of random matrix theory, a structural similarity that has
been verified to high precision numerically but is unproven. The Selberg zeta
function associated to a compact hyperbolic surface satisfies the RH analogue
unconditionally, with zeros at s = 1/2 + itⱼ where 1/4 + tⱼ² are Laplacian
eigenvalues on the surface.


§3.1  Connes's Trace Formula Approach
────────────────────────────────────────────────────────────────────────────────

The most developed spectral program is Connes's formulation on the adèle class
space X_K = 𝔸_K / K×. The spectral realization (Connes 1999) produces the
critical zeros as an absorption spectrum: the Weil explicit formula takes the form
of a Lefschetz trace formula for the action of the idèle class group C_K on the
Hilbert space L²_δ(𝔸_K/K×)₀. In this Hilbert space setting, only critical zeros
appear in the spectrum; noncritical zeros appear as resonances.

The cohomological reformulation (Connes–Consani–Marcolli) passes to the cyclic
homology HC₀(D(𝒜,φ)) of the cokernel of the "restriction map" in the abelian
category of cyclic modules. In this setting all zeros contribute to the trace
formula, and RH is equivalent to the positivity of the trace pairing

    ⟨f, f♯⟩_{H¹} ≥ 0   ∀ f ∈ 𝒮(C_K),

where f♯ = Δ⁻¹f* is the involution. This mirrors the two-step structure of the
Weil proof for function fields: (1) explicit formula as trace formula, (2)
positivity Tr(Z ⋆ Z′) > 0 for nontrivial correspondences. In the function field
case this positivity is proved via the Riemann–Roch theorem making correspondences
effective; the analogue for number fields — identifying the correct class of
"principal correspondences" — remains open.

The Connes–Consani "Riemann–Roch strategy" (AIM 2018) reformulates RH as the
existence of a Riemann–Roch formula

    dim H⁰ − dim H¹ + dim H² = ½ D·D

on the square of the scaling space, with negativity of the intersection equivalent
to RH. The definition of H¹ in this framework is currently an open problem.


§4  GEOMETRIC AND COHOMOLOGICAL APPROACHES
────────────────────────────────────────────────────────────────────────────────

Weil proved RH for curves over finite fields using intersection theory on the
surface C × C. For a curve C/𝔽_q of genus g, the key step is showing that for
any nontrivial correspondence Z ∈ P(C) (divisors modulo trivial and principal
ones),

    Tr(Z ⋆ Z′) = d(Z) + d′(Z) − Z·Δ > 0.

This is established by using the Riemann–Roch theorem to make Z effective (adjust
degree by trivial correspondences), then applying the Castelnuovo–Severi
inequality

    def(D) := 2d₁d₂ − D² ≥ 0

for divisors D on C₁ × C₂. The positivity forces all eigenvalues λⱼ of Frobenius
on H¹_ét(C̄, ℚ_ℓ) to satisfy |λⱼ| = √q, which is the Riemann hypothesis. The
quadratic form

    Tr(Z_{n,m} ⋆ Z′_{n,m}) = 2g m² + 2(1+q−N)mn + 2gq n²

being nonnegative for all integer pairs (n,m) is the discriminant condition
(1+q−N)² ≤ 4g²q, i.e., |N − q − 1| ≤ 2g√q.

Deninger has proposed a conjectural cohomology theory for arithmetic schemes X
over Spec(ℤ), modeled on foliation cohomology of a dynamical system whose "closed
orbits" correspond to primes. In this framework, a Lefschetz trace formula of the
form

    ζ(X,s) = det∞(s/2π − Θ | H¹) / [ det∞(s/2π − Θ | H⁰) · det∞(s/2π − Θ | H²) ]

would hold, where Θ is the generator of the ℝ-action and det∞ is a
zeta-regularized determinant. RH would then follow from eigenvalues of Θ on H¹
having real part 1/2, analogous to |λⱼ| = √q in the function field case. This
cohomology is conjectural; the "standard conjectures" analogue needed to run the
positivity argument unconditionally remains unproved.


§5  THE ISOMETRIC SECTION PROBLEM: STRUCTURAL LOCATION AND PRECISION OF ANALOGY
────────────────────────────────────────────────────────────────────────────────

The proposed connection runs through the spectral/operator-theoretic line. The
relevant structural parallel is between two instances of the same abstract
problem: a positivity condition on a quadratic form whose failure obstructs global
construction.

In the isometric section setting, the Killing form B on 𝔰𝔩(n,ℂ) is indefinite;
the Cartan decomposition 𝔰𝔩(n,ℂ) = 𝔰𝔲(n) ⊕ 𝔭 gives a signature (n²−1, 0) on
𝔰𝔲(n) (negative-definite restricted Killing) and opposite sign on 𝔭. The
condition for a right-invariant metric g on G = SL(n,ℂ) to admit an isometric
section through the identity — that g_e + B_k|_{𝔰𝔲(n)} be positive semidefinite
— is a quadratic positivity condition on a bilinear form over the Lie algebra. The
obstruction to globalizing this from the identity to a smooth section is the
integrability condition (bracket-closure / flatness of the associated connection),
which generically fails independently of whether the pointwise condition holds.

In Connes's formulation, RH is the positivity W(f ⋆ f♯) ≥ 0 for all test
functions f in the Schwartz space of C_K that vanish on the finite set F of zeros
forced by the explicit formula. More precisely, following Connes–Consani, it is
the positivity of the trace pairing on H¹, which is the noncommutative-geometry
analogue of the Weil condition Tr(Z ⋆ Z′) > 0. The passage from local/finite test
function positivity to global positivity on all of 𝒮(C_K) is exactly the point
where the proof is incomplete: in the archimedean case (a single place),
Connes–Consani (2020) proved a strong inequality showing the Weil local
contribution W_∞ exceeds the Sonin trace by a positive term, providing evidence
toward Weil positivity but not establishing it in full generality.

The analogy is therefore precise at the level of logical shape: in both cases one
has (i) a bilinear/quadratic form on a function space or Lie algebra, (ii) a
positivity condition on that form that is equivalent to (or implies) the global
object of interest, and (iii) a local-to-global step that constitutes the hard
content of the problem. The Hasse–Minkowski theorem for quadratic forms over ℚ
provides the canonical case where "local positivity everywhere implies global
positivity" succeeds; Weil positivity in the number field case is a problem of the
same logical type but over a function space, not a finite-dimensional vector space.


§5.1  What Would Constitute an Actual Equivalence
────────────────────────────────────────────────────────────────────────────────

Promoting this structural analogy to a theorem requires:

  1. An explicit group G: The natural candidate from the Connes program is a group
     acting on the adèle class space X_ℚ = 𝔸_ℚ / ℚ×, specifically the idèle
     class group C_ℚ = 𝔸×_ℚ / ℚ× or an extension thereof. This is not a
     semisimple Lie group of the SL(n,ℂ) type, however; it is a locally compact
     abelian group.

  2. An explicit right-invariant metric g encoding arithmetic data: The inner
     product on L²_δ(X_ℚ) used in Connes's spectral realization is determined by
     the weight δ, which in turn encodes the archimedean place. The metric would
     need to be constructed so that positive semidefiniteness of
     g_e + B_k|_{𝔰𝔲(n)} is equivalent to a statement about the spectrum of the
     scaling action.

  3. An explicit identification of the section obstruction with a zero off the
     critical line: In the correspondence Tr(Z ⋆ Z′) > 0 ⟺ |λⱼ| = √q, the
     failure of positivity is pinned to specific Frobenius eigenvalues. The
     analogous identification would require showing that failure of positive
     semidefiniteness of the metric restriction corresponds to a specific
     eigenvalue of the scaling operator lying off the line ℜ(s) = 1/2.

Without items (1)–(3) made explicit and verified, the parallel is a structural
analogy, not a theorem. The gap is not terminological but substantive: "same shape
of argument" and "logically equivalent" differ by the entire content of the
construction. The Weil proof for curves succeeds precisely because steps (1)–(3)
have concrete realizations — the curve C, the Jacobian pairing, and Frobenius
eigenvalues — that can be handled by Riemann–Roch. The number field case lacks a
proved analogue of Riemann–Roch on the adèle class space.
