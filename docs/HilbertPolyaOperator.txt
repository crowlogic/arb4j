Theorem 0 (Unitary time-change by total variation).

Let g : [0,вҲһ) вҶ’ в„қ be CВ№ with derivative gвҖІ locally integrable
and with discrete critical set

    рқ’һ = { t вүҘ 0 : gвҖІ(t) = 0 },   рқ’һ вҲ© [0,L] finite for every L > 0.

Assume вҲ«вӮҖ^вҲһ вҲЈgвҖІ(u)вҲЈ du = вҲһ. Define the total variation function

    G(t) = вҲ«вӮҖбө— вҲЈgвҖІ(u)вҲЈ du.

    (U_G f)(t) = вҲҡвҲЈgвҖІ(t)вҲЈ В· f(G(t)),

Then:

(1) G is a strictly increasing homeomorphism [0,вҲһ) вҶ’ [0,вҲһ).

(2) U_G is a local LВІ-isometry: for every compact K вҠӮ [0,вҲһ),

        вҲ«_K вҲЈ(U_G f)(t)вҲЈВІ dt = вҲ«_{G(K)} вҲЈf(T)вҲЈВІ dT.

(3) U_GвҒ»В№ U_G = id and U_G U_GвҒ»В№ = id on LВІ_loc([0,вҲһ)).

(4) For every tвӮҖ вҲҲ рқ’һ and every continuous h вҲҲ LВІ_loc([0,вҲһ)),

        lim_{t вҶ’ tвӮҖ} (U_G(U_GвҒ»В№ h))(t) = h(tвӮҖ).

Proof. See the Space file "Verification of Unitary Time-Change
Theorem."  вҲҺ

Theorem 1 (Spectral representation).

The stationary pullback of О¶ along ОЁ is

    X(T) вү” (U_ОЁвҒ»В№ О¶)(T) = О¶(ВҪ + iОЁвҒ»В№(T)) / вҲҡОЁвҖІ(ОЁвҒ»В№(T)),

The shift operator U_П„ : X(T) вҶҰ X(T+П„) extends to a strongly

The pullback X(T) and the recovery О¶(ВҪ+it) = вҲҡОЁвҖІ(t) В· X(ОЁ(t))

Theorem 2 (Translation generator D вҖ” continuous spectrum).

The infinitesimal generator of {U_П„} on PW_{[вҲ’2,0]} is

    D_T = вҲ’i d/dT,   dom(D_T) = PW_{[вҲ’2,0]}.

D_T is self-adjoint with Пғ(D_T) = [вҲ’2,0], purely absolutely
continuous, and has no eigenvalues. In t-coordinates, the unitarily
equivalent generator is D = вҲ’i ОЁвҖІ(t)вҒ»В№ d/dt on H_K, the pullback of
PW_{[вҲ’2,0]}. The Riemann zeros t_n are singularities of the
coefficient 1/ОЁвҖІ(t_n): domain constraints forcing every f вҲҲ dom(D)
to vanish at each t_n.

Proof.

The Fourier transform в„ұ : PW_{[вҲ’2,0]} вҶ’ LВІ([вҲ’2,0]) carries D_T to
the multiplication operator (M_Пү fМӮ)(Пү) = Пү fМӮ(Пү). The spectrum of
M_Пү on LВІ([вҲ’2,0]) is the essential range of Пү вҶҰ Пү on [вҲ’2,0], which
is [вҲ’2,0]. The spectral measure is Lebesgue measure dПү: no atoms, no
singular continuous part. The eigenvalue equation D_T F = О»F gives
F(T) = Ce^{iО»T}, whose Fourier transform CОҙ(ПүвҲ’О») is not in
LВІ([вҲ’2,0]), so F вҲү PW_{[вҲ’2,0]} unless C = 0.

The isometry W : LВІ(в„қ,dT) вҶ’ LВІ((0,вҲһ),dОЁ) given by (Wf)(t) = f(ОЁ(t))
intertwines U_П„ and the warped shift

    V_П„ f(t) = вҲҡ(ОЁвҖІ(t)/ОЁвҖІ(t_П„)) В· f(t_П„),

where t_П„ = ОЁвҒ»В№(ОЁ(t)+П„). Computing the generator:

    lim_{П„вҶ’0} (V_П„ f вҲ’ f)/П„ (t) = fвҖІ(t)/ОЁвҖІ(t),

so D = вҲ’i ОЁвҖІ(t)вҒ»В№ d/dt. At a zero t_n, ОЁвҖІ(t_n) = 0, so for
Df вҲҲ LВІ(dОЁ) the product ОЁвҖІ(t)вҒ»В№ fвҖІ(t) must remain locally
integrable, requiring fвҖІ(t_n) = 0. Combined with f(t_n) = 0 (from
the pullback structure), every element of dom(D) vanishes at each t_n.

Part (4) of Theorem 0 guarantees that the round-trip U_ОЁ U_ОЁвҒ»В№
recovers continuous functions even at the critical points. So D is
well-defined as a self-adjoint operator on H_K despite the vanishing
of ОЁвҖІ at the zeros: the singularity is removable in the LВІ sense.  вҲҺ

Theorem 3 (HilbertвҖ“PГіlya operator H вҖ” Riemann zeros as eigenvalues).

Define H on рқ’ҹ вү” span{K_П„ : П„ вҲҲ рқ’ө} by

where K_П„(T) = sinc(T вҲ’ П„) is the reproducing kernel of PW_{[вҲ’2,0]}
at П„, and рқ’ө = { П„ вҲҲ в„қ : X(П„) = 0 } = { ОЁ(t_n) : О¶(ВҪ+it_n) = 0 }.

Proof.

For x = вҲ‘_П„ a_П„ K_П„, y = вҲ‘_Пғ b_Пғ K_Пғ:

    вҹЁHx, yвҹ© = вҲ‘_{П„,Пғ} a_П„ bМ„_Пғ В· П„ В· вҹЁK_П„, K_Пғвҹ©,

Theorem 4 (RadonвҖ“Nikodym derivative under the shift operator).

Let Оі be the Borel measure dОі(t) = ОЁвҖІ(t) dt. For h вҲҲ в„қ, the
RadonвҖ“Nikodym derivative of T_h Оі with respect to Оі is

    d(T_h Оі)/dОі (t) = ОЁвҖІ(tвҲ’h)/ОЁвҖІ(t)

for all t with ОЁвҖІ(t) вү  0. The measures Оі and T_h Оі are equivalent
modulo null sets.

Proof.

Both Оі and T_h Оі are absolutely continuous with respect to Lebesgue
measure О», with continuous densities ОЁвҖІ(t) and ОЁвҖІ(tвҲ’h). The chain
rule gives d(T_h Оі)/dОі = ОЁвҖІ(tвҲ’h)/ОЁвҖІ(t). The zeros of ОЁвҖІ belong to
the critical set рқ’һ, which is discrete (hence countable, hence О»-null,
hence Оі-null). The same holds for ОЁвҖІ(tвҲ’h). So Оі and T_h Оі are
equivalent modulo null sets.  вҲҺ

Theorem 5 (The tangent space).

(a) In T-coordinates, the CameronвҖ“Martin space of the spectral
    structure is PW_{[вҲ’2,0]} itself.

(b) In t-coordinates, the CameronвҖ“Martin space is

        H_K = { f : f(t) = вҲҡОЁвҖІ(t) В· F(ОЁ(t)),  F вҲҲ PW_{[вҲ’2,0]} }
            = im(U_ОЁ)|_{PW_{[вҲ’2,0]}}.

(c) The infinitesimal generator of the ОЁ-translation flow, viewed
    as a vector field on (0,вҲһ), is

        Оҫ(t) = ОЁвҖІ(t)вҒ»В№ вҲӮ/вҲӮt.

    At each t, Оҫ(t) вҲҲ T_t(0,вҲһ) is the tangent vector to the curve
    П„ вҶҰ ОЁвҒ»В№(ОЁ(t)+П„) at П„ = 0. In T-coordinates, Оҫ = вҲӮ/вҲӮT.

Proof.

(a) The covariance operator C_T on LВІ(в„қ) in T-coordinates is the
orthogonal projection onto PW_{[вҲ’2,0]}. Since C_T is a projection,
C_TВІ = C_T, hence C_T^{1/2} = C_T, and the CameronвҖ“Martin space
C_T^{1/2}(LВІ(в„қ)) = PW_{[вҲ’2,0]}.

(b) The unitary operator U_ОЁ carries PW_{[вҲ’2,0]} to H_K by
definition: (U_ОЁ F)(t) = вҲҡОЁвҖІ(t) В· F(ОЁ(t)). The inner product on
H_K inherited from PW_{[вҲ’2,0]} is

    вҹЁf,gвҹ©_{H_K} = вҲ«вӮҖ^вҲһ [f(t)/вҲҡОЁвҖІ(t)] В· [g(t)/вҲҡОЁвҖІ(t)] В· ОЁвҖІ(t) dt
                 = вҲ«_в„қ F(T) G(T) dT.

(c) The flow of V_П„ sends t to t_П„ = ОЁвҒ»В№(ОЁ(t)+П„). Differentiating
at П„ = 0:

    dt_П„/dП„ |_{П„=0} = 1/ОЁвҖІ(ОЁвҒ»В№(ОЁ(t))) = 1/ОЁвҖІ(t).

This is the coefficient of вҲӮ/вҲӮt, so Оҫ(t) = ОЁвҖІ(t)вҒ»В№ вҲӮ/вҲӮt. Under
