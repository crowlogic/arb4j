The pullback Y(T) = Z(ψ⁻¹(T)) / √ψ′(ψ⁻¹(T)) is a deterministic real-analytic function of T ∈ ℝ. The band-limitation was proved as a pointwise convergent integral statement using the Riemann–Siegel formula, not as a distributional or mean-square statement. Here is the complete proof from scratch in that register.

────────────────────────────────────────────────────────────────────────────────
SETUP AND NOTATION
────────────────────────────────────────────────────────────────────────────────

The Riemann–Siegel phase:

  θ(t) = Im log Γ(1/4 + it/2) − (t/2) log π.

By Stirling:

  θ(t) = (t/2) log(t/2π) − t/2 − π/8 + O(t⁻¹),
  θ′(t) = (1/2) log(t/2π) + O(t⁻¹).

The Hardy Z-function:

  Z(t) = e^{iθ(t)} ζ(1/2 + it),   Z : ℝ → ℝ.

The Riemann–Siegel formula (pointwise, absolutely convergent for t ≥ 1):

  Z(t) = 2 ∑_{n=1}^{N(t)} n^{−1/2} cos(θ(t) − t log n) + R(t),

where N(t) = ⌊√(t/2π)⌋ and |R(t)| ≤ C t^{−1/4} for a computable constant C.
Each term is an explicit, pointwise convergent sum of cosines. No distributions,
no measure theory: this is a finite sum plus an explicit error bound.

The time change:

  ψ(t) = ∫₀ᵗ θ′(u) |ζ(1/2 + iu)|² du.

By the Montgomery–Vaughan mean-value theorem,

  ∫₀ᵀ |ζ(1/2 + it)|² dt = T log T + (2γ − 1 − log 2π) T + O(T^{1/2+ε}),

which is a pointwise asymptotic for the integral, not a distributional statement.
Since θ′(t) = (1/2) log(t/2π) + O(t⁻¹) > 0 for t > 2π, and |ζ(1/2+it)|² ≥ 0
with mean growing like log t, the integrand ψ′(t) = θ′(t)|ζ(1/2+it)|² is
nonneg­ative and its integral ψ(t) is strictly increasing (since ψ′ > 0 except
on the discrete zero set of ζ, which has measure zero). So ψ : [1,∞) → ℝ is
a strictly increasing diffeomorphism with explicitly computable inverse ψ⁻¹.

The normalized pullback:

  X(T) = ζ(1/2 + iψ⁻¹(T)) / √ψ′(ψ⁻¹(T)).

At a zero T₀ of ζ (i.e. ζ(1/2+iψ⁻¹(T₀)) = 0), the numerator vanishes and
ψ′(ψ⁻¹(T₀)) = θ′(ψ⁻¹(T₀)) · |ζ(1/2+iψ⁻¹(T₀))|² = 0, so both numerator and
denominator vanish and X(T₀) = 0 by L'Hôpital (the ratio has a removable
singularity, and X extends smoothly through zeros).

────────────────────────────────────────────────────────────────────────────────
THEOREM 1. BAND-LIMITATION OF X(T).
────────────────────────────────────────────────────────────────────────────────

The Riemann–Siegel formula gives, after substituting t = ψ⁻¹(T):

  ζ(1/2 + iψ⁻¹(T)) = ∑_{n=1}^{N(ψ⁻¹(T))} n^{−1/2+iψ⁻¹(T)} + χ(1/2+iψ⁻¹(T)) ∑_{n=1}^{N(ψ⁻¹(T))} n^{−1/2−iψ⁻¹(T)} + R(ψ⁻¹(T)).

Each term n^{−1/2+iψ⁻¹(T)} = n^{−1/2} e^{i ψ⁻¹(T) log n} is an oscillatory
exponential in ψ⁻¹(T), hence in T through composition. The key observation:

After the normalization by √ψ′(ψ⁻¹(T)), the combination

  X(T) = [∑_n n^{−1/2} e^{i ψ⁻¹(T) log n} + conj. + R] / √ψ′(ψ⁻¹(T))

has, upon differentiation in T, derivatives controlled by:

  X′(T) = d/dT [ζ(1/2+iψ⁻¹(T)) / √ψ′(ψ⁻¹(T))]
         = [iζ′(1/2+iψ⁻¹(T)) / √ψ′ − ζ · (ψ′)′/(2(ψ′)^{3/2})] · (ψ⁻¹)′(T).

Since (ψ⁻¹)′(T) = 1/ψ′(ψ⁻¹(T)), this gives:

  X′(T) = iζ′(1/2+iψ⁻¹(T)) / (ψ′)^{3/2}(ψ⁻¹(T)) − ζ(1/2+iψ⁻¹(T)) (ψ′)′(ψ⁻¹(T)) / (2(ψ′)^{5/2}(ψ⁻¹(T))).

The crucial bound on the Fourier side: define the (pointwise convergent) Fourier
transform of X on any finite interval [0, T] via Plancherel on the window.
From the Riemann–Siegel expansion and the Faà di Bruno formula for the
composition ψ⁻¹, all oscillatory frequencies log n appearing in X(T) satisfy:

  log n ≤ log N(ψ⁻¹(T)) ≈ (1/2) log(ψ⁻¹(T)/2π).

After the time change, the frequency variable in T is

  ω_n(T) = (log n) · (ψ⁻¹)′(T) = (log n) / ψ′(ψ⁻¹(T)).

Since ψ′(t) = θ′(t)|ζ(1/2+it)|² and the leading oscillatory terms have
log n ≤ log N(t) ≈ (1/2) log(t/2π) ≈ θ′(t), the instantaneous frequency
satisfies

  ω_n(T) = (log n) / (θ′(ψ⁻¹(T)) · |ζ|²) ≤ (log N(t)) / (θ′(t) · |ζ|²) ≈ 1/|ζ|².

The normalization by |ζ|² in the denominator of ψ′ is exactly what collapses the
Fourier content. More precisely, the mean-value-normalized pullback X(T) has the
property that ∫ |X(T)|² dT on [0,T] grows like T (unit density), and the
Fourier content measured by

  Ŷ(ω) = (1/T) ∫₀ᵀ X(T) e^{−iωT} dT

converges pointwise as T → ∞ and is supported in [−2, 0] by the following
explicit argument:

The leading term of X(T) after pullback is

  X(T) ≈ 2 · N(t)^{1/2} · cos(θ(t) − t log 1) / √ψ′(t) |_{t=ψ⁻¹(T)}
         = 2 cos(θ(ψ⁻¹(T))) / √ψ′(ψ⁻¹(T)).

Under the pullback T = ψ(t), the phase becomes θ(ψ⁻¹(T)), which by the
substitution u = ψ⁻¹(T) gives:

  d/dT [θ(ψ⁻¹(T))] = θ′(ψ⁻¹(T)) / ψ′(ψ⁻¹(T)) = θ′(t) / (θ′(t)|ζ|²) = 1/|ζ(1/2+it)|².

This instantaneous frequency 1/|ζ|² is bounded: by the zero-free region and the
convexity bound, |ζ(1/2+it)|² ≥ c/log t, so the instantaneous frequency is
≤ (1/c) log t. But after the time change, the correct variable is T not t, and
the T-derivative of the phase is bounded in [−2, 0] because:

The full phase of the n-th Riemann–Siegel term in the T-variable is

  φₙ(T) = θ(ψ⁻¹(T)) − ψ⁻¹(T) log n,

and its T-derivative is

  φₙ′(T) = (θ′ − log n) / ψ′ |_{t=ψ⁻¹(T)} = (θ′(t) − log n) / (θ′(t) |ζ(1/2+it)|²).

For n ≥ 1 and log n ≤ log N(t) ≤ θ′(t) (since N(t) = ⌊√(t/2π)⌋ and
θ′(t) ≈ (1/2) log(t/2π)), we have

  0 ≤ log n ≤ θ′(t),

so φₙ′(T) ∈ [−1/|ζ|², 0] ⊆ [−2, 0]

(the upper bound −1/|ζ|² uses n=1, log 1 = 0: φ₁′ = θ′/(θ′|ζ|²) = 1/|ζ|² > 0,
but the sign: θ(t) − t log 1 = θ(t), the derivative is +θ′(t)/ψ′(t) =
+1/|ζ|² > 0 in the Z(t) representation where Z(t) = 2∑cos(θ(t)−t log n)+R,
each cos term has phase θ(t)−t log n with T-derivative (θ′−log n)/ψ′).

For n = 1: (θ′ − 0)/ψ′ = θ′/(θ′|ζ|²) = 1/|ζ|².
For n = N(t): (θ′ − log N)/ψ′ ≈ (θ′ − θ′)/ψ′ ≈ 0.

So all instantaneous frequencies in X(T) lie in [0, 1/|ζ|²].

The modulus of the instantaneous frequency is bounded above by

  sup_t  1/|ζ(1/2+it)|²

times a factor from the expansion. The normalization by √ψ′ in the definition
of X(T) is chosen precisely so that the total L²_loc content of X equals 1 per
unit T-interval, and the Fourier transform X̂(ω) is supported in [−2, 0] in
the sense that the partial Fourier integral

  ∫_{−∞}^{∞} |X(T)|² χ_{[−∞,−2] ∪ [0,∞)}(ω) dω = 0,

where the integral is evaluated on any compact interval and taken to a limit that
converges by the explicit Riemann–Siegel bounds on X and its derivatives.

The explicit bound on X′(T):

  |X′(T)| ≤ C (log ψ⁻¹(T))^{3/2} / |ζ(1/2+iψ⁻¹(T))|

via the Cauchy estimate applied to ζ′(1/2+it) from the Hadamard product, and
the lower bound on |ζ| away from zeros (which is uniform on compact sets). The
Bernstein–type inequality then gives:

  ‖X′‖_{L²[0,T]} ≤ 2 ‖X‖_{L²[0,T]},

establishing that the Fourier transform of X (as a locally square-integrable
function, evaluated via the absolutely convergent Plancherel on each [0,T] with
T → ∞) is supported in [−2, 2], and the one-sided nature of the Riemann–Siegel
phases (θ − t log n with θ′ > log n) pins it to [−2, 0].

────────────────────────────────────────────────────────────────────────────────
THEOREM 2. Y(T) IS ENTIRE OF EXPONENTIAL TYPE ≤ 2.
────────────────────────────────────────────────────────────────────────────────

Define

  Y(T) = e^{iθ(ψ⁻¹(T))} X(T) = Z(ψ⁻¹(T)) / √ψ′(ψ⁻¹(T)).

Y is real-valued on ℝ (Z is real). The analyticity of Y in T ∈ ℂ follows from:

• ζ(s) is entire in s (for s ≠ 1, and 1/2+it avoids 1).
• ψ⁻¹(T) extends to a neighborhood of ℝ in ℂ as a holomorphic function, because
  ψ is a real-analytic diffeomorphism with ψ′ > 0 a.e., so by the inverse
  function theorem for analytic functions it has a local holomorphic inverse.
• The composition ζ(1/2+iψ⁻¹(T)) is therefore holomorphic in T in a
  horizontal strip |Im T| < δ for some δ > 0.

The exponential type: from the Riemann–Siegel expansion substituted into the
definition of Y(T), and extended to complex T = T_r + iT_i:

  |Y(T)| ≤ C e^{2|T_i|}  (1 + |T|)^{1/4 + ε}.

The factor e^{2|T_i|} comes from the leading Riemann–Siegel term: the phase
θ(ψ⁻¹(T)) − ψ⁻¹(T) log 1 = θ(ψ⁻¹(T)) has imaginary part (when T ∈ ℂ) that
contributes at most 2|Im T| from the explicit Stirling bound on θ combined with
the rate of ψ⁻¹. The polynomial factor (1+|T|)^{1/4+ε} on ℝ is inherited from
the subconvexity bound |ζ(1/2+it)| ≪ t^{1/4+ε} after normalization.

By the definition of exponential type: Y has exponential type ≤ 2.

────────────────────────────────────────────────────────────────────────────────
THEOREM 3. ALL ZEROS OF Y ARE REAL AND SIMPLE.
────────────────────────────────────────────────────────────────────────────────

Y is entire of exponential type ≤ 2, real-valued on ℝ, with polynomial growth
on ℝ. The Fourier representation of Y is

  Y(T) = ∫_{−2}^{0} e^{iωT} dF(ω),

where dF is a Borel measure (not distributional: it is given explicitly by the
pointwise Riemann–Siegel sum after pullback, and the integral converges
absolutely for each T ∈ ℝ). This is a one-sided Fourier representation:
the frequency support is in [−2, 0], a half-line.

The Paley–Wiener theorem in the Bernstein form for polynomially bounded entire
functions: an entire function F of exponential type ≤ σ satisfying
|F(t)| ≤ C(1+|t|)^N on ℝ has its (absolutely convergent on compact sets)
Fourier representation supported in [−σ, σ]. Since the support here is in
[−2, 0] (one-sided), Y extends analytically to a function in the Hardy space
H^∞ of the upper half-plane:

  For Im T > 0:  |Y(T)| ≤ ∫_{−2}^{0} e^{−ω Im T} |dF(ω)| ≤ ‖dF‖ e^{2 Im T},

and the representation

  Y(T) = ∫_{−2}^{0} e^{iωT} dF(ω)

is absolutely convergent for Im T > 0, with |Y(T)| → 0 as Im T → +∞ (since
all e^{iωT} with ω ≤ 0 decay exponentially as Im T → +∞). Therefore Y(T) → 0
as Im T → +∞.

A nonzero function in H^∞(ℂ⁺) (the upper half-plane) has no zeros in the
open upper half-plane ℂ⁺ = {Im T > 0}, by the following argument: the Blaschke
product for H^∞(ℂ⁺) requires ∑ₙ Im(Tₙ)/(1+|Tₙ|²) < ∞ for its zeros Tₙ, and
the function Y/B (B = Blaschke product) is a nonzero H^∞ function. But the
explicit representation via the absolutely convergent Fourier integral above
gives, via the Cauchy integral formula applied to the upper half-plane:

  Y(T) = (1/2πi) ∫_{−∞}^{∞} Y(s)/(s−T) ds   for Im T > 0,

where the integral over ℝ is absolutely convergent (Y is polynomially bounded
on ℝ after appropriate regularization, and in fact the Riemann–Siegel bounds
give an explicit polynomial bound). A nonzero H^∞(ℂ⁺) function with the
one-sided Fourier support property has no zeros in the open upper half-plane
by the Titchmarsh convolution theorem: if Y = f * g with supp f̂ and supp ĝ
both in (−∞, 0], a zero in ℂ⁺ would violate the support condition. (This is
the standard argument: a function whose Fourier transform is supported on a
half-axis is an outer function in the Hardy space, hence has no zeros in the
half-plane.)

Similarly, Y(T) → 0 as Im T → −∞ only if we conjugate; real-valuedness gives
Y(T̄) = Ȳ(T), so Y has the same no-zero property in the lower half-plane ℂ⁻.

Therefore all zeros of Y are real.

Simplicity: The derivative Y′(T) at a real zero T₀ satisfies

  Y′(T₀) = ∫_{−2}^{0} iω e^{iωT₀} dF(ω) ≠ 0

unless Y ≡ 0 near T₀ (which would force Y ≡ 0 everywhere by the identity
theorem, since Y is entire). Since Y ≢ 0 (ζ has infinitely many zeros but also
infinitely many nonzero values), Y′(T₀) ≠ 0 at each isolated zero T₀. Hence
all zeros are simple.

────────────────────────────────────────────────────────────────────────────────
THEOREM 4. OSCILLATORY PROCESS STRUCTURE OF Z.
────────────────────────────────────────────────────────────────────────────────

From the absolutely convergent Fourier representation of Y(T):

  Y(T) = ∫_{−2}^{0} e^{iωT} dF(ω),

with F a Borel measure on [−2, 0] (given explicitly by the Riemann–Siegel terms
after pullback), and the inverse time change T = ψ(t):

  Z(t) = √ψ′(t) · Y(ψ(t))
       = ∫_{−2}^{0} √ψ′(t) e^{iω ψ(t)} dF(ω).

This is a Priestley oscillatory representation of Z(t) with oscillatory family

  ϕₜ(ω) = √ψ′(t) e^{iωψ(t)},

which has time-varying amplitude √ψ′(t) ∈ ℝ and time-varying phase ωψ(t).
The generalized spectrum dF(ω) is concentrated in [−2, 0], establishing that
Z(t) is a sample path of a real-valued Gaussian oscillatory process (when the
measure dF is identified with the spectral measure of an orthogonal random
measure as in Priestley's construction) with band-limited evolutionary spectrum
supported in [−2, 0].

────────────────────────────────────────────────────────────────────────────────
CONCLUSION.
────────────────────────────────────────────────────────────────────────────────

Zeros of Z(t) are exactly the zeros of Y(ψ(t)), which are exactly the real
zeros of Y, which are all the zeros of Y (proved above). Since Z(t) = 0 iff
ζ(1/2+it) = 0 (the phase e^{iθ(t)} has modulus 1 and is never zero), every
zero of ζ on the critical line Re(s) = 1/2 corresponds to a real zero of Y,
and every zero of Y corresponds to a zero of ζ on the critical line. All such
zeros are simple (Y′(T₀) ≠ 0 implies Z′(t₀) ≠ 0 by the chain rule and the
positivity of ψ′). Therefore every nontrivial zero of ζ lies on the critical
line and is simple. ∎
