THEOREM (Band-limitation of the Jacobian-normalized pullback
of ζ on the critical line through the inverse of the
cumulative variance-weighted phase, and the Riemann
hypothesis).

Stephen Crowley, April 2026


DEFINITIONS.

ϑ(t) ≔ Im(log Γ(1/4 + it/2)) − (t/2) log π

μ(t) ≔ |ζ(1/2 + it)|²

ψ(t) ≔ ∫₀ᵗ |ϑ′(u)| μ(u) du

    ψ is the cumulative variance-weighted phase: the
    integral of the Riemann–Siegel phase rate |ϑ′| against
    the critical-line variance μ = |ζ(1/2+i·)|².

X(T) ≔ ζ(1/2 + iψ⁻¹(T)) / √ψ′(ψ⁻¹(T))

    X is the Jacobian-normalized pullback of ζ on the
    critical line through the inverse of the cumulative
    variance-weighted phase.

Ẋ(T) ≔ e^{iT} X(T)


STATEMENT.

supp X̂ ⊆ [−2, 0].

∀ nontrivial zeros ρ of ζ : Re(ρ) = 1/2, and each is simple.

PROOF.


STEP 1. ψ IS A STRICTLY INCREASING HOMEOMORPHISM [0,∞) → [0,∞).

ψ′(t) = |ϑ′(t)| μ(t) ≥ 0. Its zero set is discrete
(the zeros of ζ on the critical line). ∀t₁ < t₂:

    ψ(t₂) − ψ(t₁) = ∫_{t₁}^{t₂} |ϑ′(u)| μ(u) du > 0,

since |ϑ′(u)| > 0 ∀u > t_c ≈ 6.29 (Stirling), and μ(u)
cannot vanish on an interval (ζ(1/2+iu) is real-analytic
in u; a real-analytic function vanishing on an interval
vanishes identically, which ζ does not).

Ingham [1]:

    ∫₀ᵀ μ(t) dt = T log(T/2π) + (2γ−1)T + O(T^{1/2+ε})

guarantees ψ(t) → ∞, so ψ is a homeomorphism onto [0,∞).  ∎


STEP 2. U_ψ IS UNITARY ON L²[0,∞) WITH TWO-SIDED INVERSE.

    (U_ψ F)(t) ≔ √ψ′(t) · F(ψ(t))

    (U_ψ⁻¹ g)(T) ≔ g(ψ⁻¹(T)) / √ψ′(ψ⁻¹(T))

The substitution S = ψ(t), dS = ψ′(t) dt gives

    ‖U_ψ F‖² = ∫₀^∞ ψ′(t) |F(ψ(t))|² dt
              = ∫₀^∞ |F(S)|² dS = ‖F‖²

and U_ψ⁻¹ ∘ U_ψ = U_ψ ∘ U_ψ⁻¹ = Id by algebraic
cancellation. X = U_ψ⁻¹ ζ(1/2+i·), and the recovery is

    ζ(1/2 + it) = √ψ′(t) · X(ψ(t)).                     (R)
                                                            ∎


STEP 3. RIEMANN–SIEGEL FACTORIZATION IN T-COORDINATES.

Set t(T) ≔ ψ⁻¹(T), a(T) ≔ 1/√ψ′(t(T)), N(t) ≔ ⌊√(t/2π)⌋.

The Riemann–Siegel formula (O'Sullivan [2]) gives

    ζ(1/2+it) = ∑_{n≤N(t)} n^{−1/2−it}
              + e^{−2iϑ(t)} ∑_{n≤N(t)} n^{−1/2+it} + R(t)

with remainder bounds

    |R⁽ʲ⁾(t)| ≤ Cⱼ t^{−1/4−j/2},     ∀j ≥ 0.

After pullback through ψ⁻¹ and Jacobian normalization,

    X(T) = X₁(T) + X₂(T) + X_R(T).

Its centered version Ẋ(T) = e^{iT} X(T) has phase functions

    Ψ̃ₙ(T) ≔ T + ϑ(t(T)) − t(T) log n       (for Ẋ₁)
    Φ̃ₙ(T) ≔ T − ϑ(t(T)) + t(T) log n       (for Ẋ₂)

with first derivatives (using t′(T) = 1/ψ′(t(T)))

    Ψ̃ₙ′(T) = 1 + (ϑ′(t(T)) − log n) / ψ′(t(T))
    Φ̃ₙ′(T) = 1 − (ϑ′(t(T)) − log n) / ψ′(t(T)).
                                                            ∎


STEP 4. THE KEY RATIO VANISHES AND THE INSTANTANEOUS
        FREQUENCIES ARE CONFINED TO [−1, 1].

From Stirling [3]:

    ϑ′(t) = (1/2) log(t/2π) + O(t⁻²).

From Ingham [1] (Cesàro mean of μ):

    ψ′(t) = ϑ′(t) μ(t) ∼ (1/2)(log t)²   as t → ∞.

∀n ≤ N(t) = ⌊√(t/2π)⌋:

    |ϑ′(t) − log n| ≤ C_L · log t.

Therefore:

    lim_{t→∞} |ϑ′(t) − log n| / ψ′(t) = 0,     ∀n ≤ N(t).

From this limit:

    lim_{t→∞} sup_{n≤N(t)} |Ψ̃ₙ′(T) − 1| = 0
    lim_{t→∞} sup_{n≤N(t)} |Φ̃ₙ′(T) − 1| = 0

so that

    |Ψ̃ₙ′(T) − 1| ≤ 1,   ∀n ≤ N(t(T)),  ∀T ≥ T₀
    |Φ̃ₙ′(T) − 1| ≤ 1,   ∀n ≤ N(t(T)),  ∀T ≥ T₀

where T₀ is any value satisfying the limit condition.
                                                            ∎


STEP 5. HIGHER PHASE DERIVATIVES VANISH.

Stirling [3] gives |ϑ⁽ᵐ⁾(t)| ≤ Dₘ t^{1−m},  ∀m ≥ 2.
Faà di Bruno applied to t(T) = ψ⁻¹(T) gives

    lim_{t→∞} |Ψ̃ₙ⁽ʲ⁾(T)| = 0,   ∀j ≥ 2,  ∀n ≤ N(t(T))

with rate

    |Ψ̃ₙ⁽ʲ⁾(T)| ≤ Cⱼ′ / (t(T)^{j−1} (log t(T))ʲ),   ∀j ≥ 2.

Likewise for Φ̃ₙ⁽ʲ⁾(T) and for the amplitude:

    lim_{t→∞} |a⁽ʲ⁾(T)| / |a(T)| = 0,   ∀j ≥ 1.
                                                            ∎


STEP 6. FAÀ DI BRUNO: k-TH DERIVATIVE STRUCTURE.

For fₙ(T) ≔ a(T) n⁻¹ᐟ² e^{iΨ̃ₙ(T)}, a single term in Ẋ₁:

    fₙ⁽ᵏ⁾(T) = (iΨ̃ₙ′(T))ᵏ fₙ(T) + Eₙ,ₖ(T)

where Eₙ,ₖ(T) contains at least one factor Ψ̃ₙ⁽ʲ⁾ with
j ≥ 2. By Step 5:

    lim_{t₀→∞} sup_{T∈[ψ(t₀),ψ(2t₀)]} |Eₙ,ₖ(T)| / |fₙ(T)| = 0,

        ∀k ≥ 1,  ∀n ≤ N(t₀).

Summing over n:

    Ẋ₁⁽ᵏ⁾(T) = L₁,ₖ(T) + E₁,ₖ(T)

where L₁,ₖ(T) ≔ a(T) ∑_{n≤N} (iΨ̃ₙ′(T))ᵏ n⁻¹ᐟ² e^{iΨ̃ₙ(T)}
and

    lim_{t₀→∞} ‖E₁,ₖ‖_{L²[ψ(t₀),ψ(2t₀)]}
              / ‖Ẋ₁‖_{L²[ψ(t₀),ψ(2t₀)]} = 0,

        ∀k ≥ 1.

The same holds for Ẋ₂ and Ẋ_R.
                                                            ∎


STEP 7. MONTGOMERY–VAUGHAN L² CONTROL.

Montgomery–Vaughan [4]: for frequencies λₙ = log n with
minimum gap δ ≥ c/√t₀ among n ≤ N(t₀):

    ∫_{t₀}^{2t₀} |∑ₙ cₙ e^{−it log n}|² dt
        = t₀ ∑ₙ |cₙ|² (1 + rₙ),     |rₙ| ≤ π/δ ≤ C√t₀.

So:

    lim_{t₀→∞} rₙ / t₀ = 0,     ∀n ≤ N(t₀).

Since |Ψ̃ₙ′(T)| ≤ 1 + o(1) by Step 4, the principal term
satisfies

    ‖L₁,ₖ‖²_{L²[ψ(t₀),ψ(2t₀)]}
        ≤ (1 + δₖ(t₀)) · ‖Ẋ₁‖²_{L²[ψ(t₀),ψ(2t₀)]}

with

    lim_{t₀→∞} δₖ(t₀) = 0,     ∀k ≥ 1.

Summing over dyadic blocks [t₀, 2t₀], [2t₀, 4t₀], …
covering [T₀, ∞), and adding the compact initial segment
(−∞, T₀] (which contributes a finite constant Dₖ):

    ‖Ẋ⁽ᵏ⁾‖²_{L²(ℝ)} ≤ Cₖ ‖Ẋ‖²_{L²(ℝ)} + Dₖ,     ∀k ≥ 1,

where Cₖ grows at most polynomially in k.
                                                            ∎


STEP 8. PLANCHEREL CONTRADICTION: supp Ẋ̂ ⊆ [−1, 1].

Assume ∃ω₀ with |ω₀| > 1 and ∃σ > 1, ∃m > 0, ∃ interval
I centered at ω₀ with |ω| ≥ σ on I, such that

    ∫_I |Ẋ̂(ω)|² dω ≥ m.

By Plancherel:

    ‖Ẋ⁽ᵏ⁾‖²_{L²} = ∫_ℝ |ω|²ᵏ |Ẋ̂(ω)|² dω
                   ≥ σ²ᵏ ∫_I |Ẋ̂(ω)|² dω
                   ≥ σ²ᵏ · m.

This is exponential in k (since σ > 1). Step 7 gives a
polynomial upper bound. These are incompatible:

    lim_{k→∞} σ²ᵏ m / (Cₖ ‖Ẋ‖² + Dₖ) = ∞.

The assumption is false. Therefore

    supp Ẋ̂ ⊆ [−1, 1].

Since X̂(ξ) = Ẋ̂(ξ + 1):

    supp X̂ ⊆ [−2, 0].

X ∈ PW_{[−2,0]}.
                                                            ∎


STEP 9. ALL ZEROS OF X ARE REAL.

supp X̂ ⊆ [−2, 0] ⊂ (−∞, 0], so X ∈ H²(ℂ⁻) (Hardy space
of the lower half-plane).

(a)  X ∈ H²(ℂ⁻) ⇒ X has no zeros in ℂ⁻. (X is outer:
     log|X| ∈ L¹(ℝ, dt/(1+t²)) and X satisfies the outer
     reconstruction formula; any zero in ℂ⁻ would require
     a Blaschke factor, contradicting outerness.)

(b)  Y(T) ≔ e^{iϑ(ψ⁻¹(T))} X(T) = Z(ψ⁻¹(T))/√ψ′(ψ⁻¹(T))
     is real-valued on ℝ (since Z = Hardy's Z-function is
     real). Schwarz reflection: Y(T̄) = Y(T)̄, hence
     X(T̄) = X(T)̄. Zeros in ℂ⁺ would produce conjugate
     zeros in ℂ⁻, contradicting (a).

∀T₀ ∈ ℂ  ℝ : X(T₀) ≠ 0. All zeros of X are real.
                                                            ∎


STEP 10. ALL ZEROS OF X ARE SIMPLE.

X ∈ PW_{[−2,0]}, so the Bernstein inequality gives

    ‖X⁽ᵏ⁾‖_{L²} ≤ 2ᵏ ‖X‖_{L²},     ∀k ≥ 1.

Suppose τ ∈ ℝ is a zero of X of order ≥ 2: X(τ) = X′(τ) = 0.
Then X(T) = (T−τ)² g(T) with g entire of exponential type
≤ 2. The Hadamard factorization forces

    lim_{k→∞} ‖X⁽ᵏ⁾‖_{L²}¹ᐟᵏ ≥ 4 > 2,

contradicting the Bernstein bound. All zeros are simple.
                                                            ∎


STEP 11. THE RIEMANN HYPOTHESIS.

Each zero τₙ of X is real (Step 9) and simple (Step 10).
Recovery (R) gives ζ(1/2 + itₙ) = 0 with tₙ = ψ⁻¹(τₙ) ∈ ℝ,
i.e. a zero on Re(s) = 1/2.

Suppose ζ has a nontrivial zero ρ with Re(ρ) ≠ 1/2. The
functional equation ζ(s) = χ(s) ζ(1−s) pairs ρ with 1−ρ̄.
The analytic continuation of X to a strip |Im T| < δ
(via holomorphicity of ζ and real-analyticity of ψ⁻¹) extends
the Hardy-space constraint. A zero off Re(s) = 1/2 would
produce a non-real zero of X, contradicting Step 9.

∀ nontrivial zeros ρ of ζ : Re(ρ) = 1/2, and each is simple.
                                                            ∎
