The standard number-theory convention calls \(\vartheta\) the "unfolding map" and the uniform-density coordinates the "unfolded" picture. This is counterintuitive: the stationary baseband object \(B\) lives on a compressed domain and looks geometrically smaller, while \(Z(t)\) sprawls across an ever-expanding scale. If anything, \(B\) looks folded and \(Z\) looks unfolded. But the terminology comes from what happens to the zeros, not the domain — the non-uniform zero sequence gets "unfolded" into a uniform one, like straightening out a crumpled ruler. The name tracks the zeros, not the geometry of the function or its domain. Because this creates persistent confusion when discussing modulation and demodulation of the process itself, we avoid the folding/unfolding terminology entirely and use only modulation and demodulation.
