THE CHRISTOFFEL-DARBOUX KERNEL AND THE PADÉ APPROXIMANT
========================================================

All formulas link the CD kernel K_M(z,w) to the Padé approximant
Φ(M)(z) = Pn_M*(z) / Q_M*(z).


1. DEFINITIONS
--------------

Moment functional μ with moments μ_k = a_{k+1}.

Monic orthogonal polynomials Q_k (degree k), satisfying the Favard recurrence:
  Q_{k+1}(z) = (z - α_k) Q_k(z) - β_k Q_{k-1}(z),  Q_0 = 1, Q_{-1} = 0

Squared norms: h_k = ⟨Q_k, Q_k⟩_μ

Associated polynomials of the second kind Pn_k (same recurrence, Pn_0 = 0,
Pn_1 = h_0):
  Pn_{k+1}(z) = (z - α_k) Pn_k(z) - β_k Pn_{k-1}(z)

Reciprocal polynomials:
  Q_M*(z) = z^M Q_M(1/z)         (degree M, the Padé denominator)
  Pn_M*(z) = z^{M-1} Pn_M(1/z)  (degree M-1, the Padé numerator)

Padé approximant:
  Φ(M)(z) = Pn_M*(z) / Q_M*(z)

CD kernel (sum form):
  K_M(z,w) = Σ_{k=0}^{M} Q_k(z) Q_k(w) / h_k

Jacobi matrix (M×M):
        ⎡ α_0   1     0    ···   0   ⎤
        ⎢ β_1   α_1   1    ···   0   ⎥
  J_M = ⎢  0    β_2   α_2  ···   0   ⎥
        ⎢  ·     ·     ·    ·     ·   ⎥
        ⎣  0    ···  β_{M-1} α_{M-1} ⎦

Eigenvalues of J_M = roots ζ_j of Q_M.


2. FORMULA: CHRISTOFFEL-DARBOUX (closed form)
----------------------------------------------

The sum collapses to a ratio of two consecutive orthogonal polynomials:

  K_M(z,w) = [Q_{M+1}(z) Q_M(w) - Q_M(z) Q_{M+1}(w)] / [h_{M+1} (z - w)]

Confluent form (w → z):

  K_M(z,z) = [Q_{M+1}'(z) Q_M(z) - Q_M'(z) Q_{M+1}(z)] / h_{M+1}


3. FORMULA: PADÉ DENOMINATOR = RECIPROCAL OF Q_M
-------------------------------------------------

  Q_M*(z) = z^M Q_M(1/z)

The roots of Q_M*(z) are z_j = 1/ζ_j, where ζ_j are the roots of Q_M.
These are the poles of Φ(M)(z).


4. FORMULA: PARTIAL FRACTION OF THE PADÉ
-----------------------------------------

  Φ(M)(z) = Σ_{j=1}^{M} c_j / (z - z_j)

where z_j = 1/ζ_j are the poles.


5. FORMULA: RESIDUE (Golub-Welsch)
-----------------------------------

The residue at the j-th pole z_j = 1/ζ_j:

  c_j = h_{M-1} / [Q_M'(ζ_j) Q_{M-1}(ζ_j)]

This is the Golub-Welsch identity. The residue equals the Gauss quadrature
weight w_j divided by h_0, times h_{M-1}:

  w_j = h_0 / [Q_M'(ζ_j) Q_{M-1}(ζ_j)]

  c_j = (h_{M-1} / h_0) w_j


6. FORMULA: CD KERNEL DIAGONAL AT ROOTS OF Q_M
-----------------------------------------------

At roots ζ_i, ζ_j of Q_M (eigenvalues of J_M):

  K_M(ζ_i, ζ_j) = δ_{ij} / w_j

In particular, at a single root ζ_j:

  K_M(ζ_j, ζ_j) = 1 / w_j = Q_M'(ζ_j) Q_{M-1}(ζ_j) / h_0


7. FORMULA: RESIDUE IN TERMS OF THE CD KERNEL
----------------------------------------------

Substituting Formula 6 into Formula 5:

  c_j = h_{M-1} / [h_0 K_M(ζ_j, ζ_j)]

This is the exact formula linking the CD kernel to the Padé residue.
The residue of Φ(M) at its j-th pole equals h_{M-1} / h_0 divided by the
diagonal of K_M evaluated at the corresponding root of Q_M.


8. FORMULA: FULL PADÉ IN TERMS OF THE CD KERNEL
------------------------------------------------

Combining Formulas 4, 6, 7:

  Φ(M)(z) = Σ_{j=1}^{M}  h_{M-1} / [h_0 K_M(ζ_j, ζ_j)] · 1/(z - 1/ζ_j)

where ζ_j are the roots of Q_M.


9. FORMULA: PADÉ AS RESOLVENT OF THE JACOBI MATRIX
---------------------------------------------------

The Padé of the moment generating function M(w) = Σ μ_k w^k:

  Φ_M(w) = e_1^T (wI - J_M)^{-1} e_1 = Σ_{j=1}^{M} (v_j^{(1)})^2 / (w - ζ_j)

where v_j^{(1)} = e_1^T v_j is the first component of the j-th normalized
eigenvector of J_M, and ζ_j are the eigenvalues.

The Padé of the Cauchy transform C(z) = Σ μ_k / z^{k+1} is obtained by the
substitution w = 1/z:

  Φ(M)(z) = Φ_M(1/z) = Σ_{j=1}^{M} -(v_j^{(1)})^2 / [ζ_j (z - 1/ζ_j)]

The Golub-Welsch identity relates the eigenvector components to the
quadrature weights:

  (v_j^{(1)})^2 = w_j / h_0 = 1 / [Q_M'(ζ_j) Q_{M-1}(ζ_j)]


10. FORMULA: CD KERNEL AS BILINEAR EXPANSION OF CAUCHY KERNEL
-------------------------------------------------------------

The Cauchy kernel has the bilinear expansion:

  1/(z - w) = Σ_{k=0}^{∞} Q_k(z) Q_k(w) / h_k     (for |z| > |w|)

The CD kernel is the degree-M truncation:

  K_M(z,w) = Σ_{k=0}^{M} Q_k(z) Q_k(w) / h_k ≈ 1/(z - w)

The error is the tail:

  1/(z - w) - K_M(z,w) = Σ_{k=M+1}^{∞} Q_k(z) Q_k(w) / h_k


11. FORMULA: CHRISTOFFEL FUNCTION (lower bound)
------------------------------------------------

The diagonal K_M(z,z) is the Christoffel function Λ_M(z)^{-1}:

  K_M(z,z) = Σ_{k=0}^{M} Q_k(z)^2 / h_k = Λ_M(z)^{-1}

It satisfies the variational property:

  Λ_M(z) = min { ∫ P(w)^2 dμ(w) : P ∈ Π_M, P(z) = 1 }


12. FORMULA: CD KERNEL INTEGRAL
-------------------------------

For any polynomial f of degree ≤ M:

  ∫ K_M(z,w) f(w) dμ(w) = f(z)

(reproducing property: K_M projects onto Π_M)
