\begin{lemma}[{П‘вҖі > 0 on (0, вҲһ)}]\label{node-lem-П‘''}

For every n вүҘ 1, every c > cвӢҶ, and every t вүҘ 2ПҖ nВІ, define ПүвӮҷ(t) вү” (П‘вҖІ(t) вҲ’ log n) вҒ„ (П‘вҖІ(t) + c). Then ПүвӮҷ(t) вҲҲ [0, 1).
