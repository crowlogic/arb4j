The expression was

\[
\int_0^T \left|\zeta\Bigl(\tfrac12+it\Bigr)\right|^2 dt
= \int_0^T \vartheta'(t)\,\mu(t)\,\bigl|\zeta_{\rm st}(\vartheta(t))\bigr|^2 dt
= \int_{\vartheta(0)}^{\vartheta(T)} \mu\!\bigl(\vartheta^{-1}(\tau)\bigr)\,\bigl|\zeta_{\rm st}(\tau)\bigr|^2 d\tau,
\]

with

\[
\mu(t) = \log\frac{t}{2\pi} + 2\gamma + 2\sum_{n=2}^\infty \frac{d(n)}{\sqrt{n}}\cos(t\log n)
\]

and

\[
\zeta\Bigl(\tfrac12 + it\Bigr)
= \sqrt{\vartheta'(t)\,\mu(t)}\;\zeta_{\rm st}\bigl(\vartheta(t)\bigr).
\]
