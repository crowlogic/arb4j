═══════════════════════════════════════════════════════════════════════════════
THEOREM (Band-Limitation of Y under the Unitary Pullback)
═══════════════════════════════════════════════════════════════════════════════

SETUP.

Let ϑ : [t_*, ∞) → ℝ be the Riemann–Siegel theta function,

  ϑ(t) = Im log Γ(1/4 + it/2) − (t/2)·log π,

where t_* is the unique point past which ϑ'(t) > 0 strictly. Define the
time-change weight

  g'(t) = |ϑ'(t)| · |ζ(1/2 + it)|²,

and the total variation time-change function

  G(t) = ∫₀ᵗ |ϑ'(u)| · |ζ(1/2 + iu)|² du.

Write σ(τ) = G⁻¹(τ). The Unitary Time-Change (UTC) Theorem applies: since
|ϑ'(t)| > 0 for t ≥ t_* and |ζ(1/2+it)|² > 0 a.e., G is strictly increasing
with G(t) → ∞, so G is a homeomorphism [t_*, ∞) → [G(t_*), ∞) and G⁻¹ is
continuous and strictly increasing.

Define the unitary pullback

  Y(τ) = ζ(1/2 + iσ(τ)) / (|ζ(1/2 + iσ(τ))| · √|ϑ'(σ(τ))|),

so |Y(τ)| = 1/√|ϑ'(σ(τ))| a.e.

THEOREM. Y ∈ L²_loc([G(t_*), ∞))  and  supp(Ŷ) ⊆ [−2, 0].


═══════════════════════════════════════════════════════════════════════════════
PROOF
═══════════════════════════════════════════════════════════════════════════════

───────────────────────────────────────────────────────────────────────────────
STEP 1. Y ∈ L²_loc
───────────────────────────────────────────────────────────────────────────────

Fix a compact window [τ₀, 2τ₀]. By the UTC isometry with weight g'(t), the
substitution dτ = g'(t) dt gives:

  ∫_{τ₀}^{2τ₀} |Y(τ)|² dτ

    = ∫_{σ(τ₀)}^{σ(2τ₀)}
        |ζ(1/2+it)|² / (|ϑ'(t)| · |ζ(1/2+it)|²)
        · |ϑ'(t)| · |ζ(1/2+it)|² dt

    = ∫_{σ(τ₀)}^{σ(2τ₀)} |ζ(1/2+it)|² dt.               ... (L2)

By the classical mean-square bound ∫₀ᵀ |ζ(1/2+it)|² dt ~ T log T, (L2) is
finite for every compact window. Hence Y ∈ L²_loc.

Set t₀ = σ(τ₀), t₁ = σ(2τ₀), T = t₁ − t₀. The window length satisfies
T ~ t₀ (shown in Step 5), and ‖Y‖²_{L²([τ₀,2τ₀])} ~ T log T.             □


───────────────────────────────────────────────────────────────────────────────
STEP 2. RIEMANN–SIEGEL IDENTITY IN THE G-VARIABLE
───────────────────────────────────────────────────────────────────────────────

The Riemann–Siegel identity is: for t ≥ t_RS,

  ζ(1/2 + it) = S₁(t) + e^{−2iϑ(t)} · S₂(t) + R(t),          ... (RS)

where

  S₁(t) = Σ_{n≤N(t)} n^{−1/2} · e^{−it·log n},
  S₂(t) = Σ_{n≤N(t)} n^{−1/2} · e^{+it·log n},
  N(t)  = ⌊√(t/2π)⌋,

and the remainder satisfies

  |R^{(j)}(t)| ≤ C_j · t^{−1/4 − j/2}   for all j ≥ 0.       ... (Rbd)

Note: at t = σ(τ), the factor e^{−2iϑ(σ(τ))} is NOT equal to e^{−2iτ},
because the time change is G (which weights by |ζ|²), not ϑ alone.

Divide (RS) at t = σ(τ) by |ζ(1/2+it)| · √|ϑ'(t)| and write

  B(τ) := [|ϑ'(σ(τ))| · |ζ(1/2 + iσ(τ))|²]^{−1/2}.

Then:

  Y(τ) = B(τ) · S₁(σ(τ))
        + B(τ) · e^{−2iϑ(σ(τ))} · S₂(σ(τ))
        + B(τ) · R(σ(τ))
       =: F(τ) + H(τ) + E(τ).                                   ... (★)


───────────────────────────────────────────────────────────────────────────────
STEP 3. PHASE DERIVATIVES AND NON-POSITIVE INSTANTANEOUS FREQUENCIES
───────────────────────────────────────────────────────────────────────────────

Set t = σ(τ). Write σ'(τ) = 1/g'(σ(τ)) = 1/(|ϑ'(t)| · |ζ(1/2+it)|²).

FIRST SUM F. The n-th term has phase φ_{1,n}(τ) = −σ(τ) · log n, so

  ω_{1,n}(τ) = dφ_{1,n}/dτ = −log n · σ'(τ)
              = −log n / (|ϑ'(t)| · |ζ(1/2+it)|²).              ... (F1)

SECOND SUM H. The n-th term has phase

  φ_{2,n}(τ) = −2ϑ(σ(τ)) + σ(τ) · log n,

so

  ω_{2,n}(τ) = dφ_{2,n}/dτ
              = −2|ϑ'(t)| · σ'(τ) + log n · σ'(τ)
              = (−2|ϑ'(t)| + log n) / (|ϑ'(t)| · |ζ|²)
              = (−2 + log n/|ϑ'(t)|) / |ζ|².                   ... (F2)

NON-POSITIVITY. From N(t) = ⌊√(t/2π)⌋:

  log n ≤ log N(t) ≤ (1/2)·log(t/2π)   for all n ≤ N(t).       ... (NB)

The Stirling expansion of log Γ gives

  |ϑ'(t)| = (1/2)·log(t/2π) − π/(8t) + O(t^{−3}).              ... (TH)

Combining (NB) and (TH):

  log n / |ϑ'(t)| ≤ (1/2)·log(t/2π) / [(1/2)·log(t/2π) − π/(8t) + O(t^{−3})]
                   = 1 + ε(t),   ε(t) = O(1/(t log t)) → 0.    ... (BOUND)

Since log n ≥ 0 and |ϑ'(t)| > 0 and |ζ|² > 0 a.e.:

  ω_{1,n}(τ) = −log n/(|ϑ'(t)| · |ζ|²) ≤ 0.

  ω_{2,n}(τ) = (−2 + log n/|ϑ'(t)|) / |ζ|² ≤ (−2 + 1 + ε(t)) / |ζ|² ≤ 0.

Every instantaneous frequency in (★) is non-positive everywhere.            □


───────────────────────────────────────────────────────────────────────────────
STEP 4. SLOW VARIATION OF B(τ)
───────────────────────────────────────────────────────────────────────────────

Write B(τ) = [|ϑ'(σ(τ))| · |ζ(1/2+iσ(τ))|²]^{−1/2}. By the Faà di Bruno
formula, each derivative B^{(j)}(τ) is a finite sum of terms of the form

  (d^{|π|}/dt^{|π|} [|ϑ'(t)|^{−1/2} · |ζ(1/2+it)|^{−1}])|_{t=σ(τ)}
  · ∏_{B∈π} (σ)^{(|B|)}(τ),

where π ranges over set-partitions of {1,...,j}.

BOUNDS ON σ^{(k)}(τ): Differentiating G(σ(τ)) = τ repeatedly yields

  σ'   = 1/g'(t),
  σ''  = −g''(t)/g'(t)³,
  σ^{(k)} = Q_k(g'(t),...,g^{(k)}(t)) / g'(t)^{2k−1},

for universal polynomials Q_k. Since g'(t) = |ϑ'(t)| · |ζ|² ~ (log t/2) · log t,
and Paris's bounds give |ϑ^{(m)}(t)| ≤ C_m · t^{1−m}, each σ^{(k)}(τ) is
O(τ₀^{−k+ε}) relative to B(τ). Each Faà di Bruno term satisfies
|term| ≪ B(τ) · τ₀^{−1+ε}, so

  |B^{(j)}(τ)| ≪_j B(τ) · τ₀^{−1+ε}   on [τ₀, 2τ₀].          ... (SV)

The bandwidth of B on [τ₀, 2τ₀] is O(τ₀^{−1+ε}).                         □


───────────────────────────────────────────────────────────────────────────────
STEP 5. SOECTRAL RADIUS BOUND VIA MONTGOMERY–VAUGHAN
───────────────────────────────────────────────────────────────────────────────

Fix [τ₀, 2τ₀]. Set t₀ = σ(τ₀), t₁ = σ(2τ₀), N₀ = N(t₀).

WINDOW LENGTH. From G(t₁) − G(t₀) = τ₀ and g'(t) ~ |ϑ'(t₀)| · |ζ|²
averaged over [t₀, t₁], with ∫_{t₀}^{t₁} |ζ|² dt ~ T log T and
|ϑ'(t₀)| ~ (1/2) log t₀, one gets T = t₁ − t₀ ~ t₀. Hence

  N₀/T ~ √t₀/t₀ = t₀^{−1/2} → 0,   so T ≫ N₀.                ... (WIN)

K-TH DERIVATIVE. Differentiating F k times and applying (SV) to discard lower
Faà di Bruno terms (each contributes O(τ₀^{−1+ε})‖F‖), the leading term is

  F^{(k)}(τ)|_lead = B(τ) · Σ_{n≤N₀} n^{−1/2} · (ω_{1,n}(τ))^k
                     · e^{iφ_{1,n}(τ)}.

CHANGE VARIABLES τ → t via dτ = g'(t) dt:

  ‖F^{(k)}‖²_{L²([τ₀,2τ₀])}

    = ∫_{t₀}^{t₁}
        (1/(|ϑ'(t)| · |ζ|²)^{2k})
        · |Σ_{n≤N₀} n^{−1/2} · (log n)^k · e^{−it·log n}|²
        dt
      + O(τ₀^{−1+ε})‖F‖².                                      ... (CV)

FREEZE ϑ'(t) ≈ ϑ'(t₀) with relative error O(t₀^{−1}) from (TH). Set

  a_n^{(k)} = n^{−1/2} · (log n / |ϑ'(t₀)|)^k.

MONTGOMERY–VAUGHAN (1974): For distinct frequencies λ_n = −log n with gap
δ ≥ 1/(N₀+1) and window T satisfying T ≫ π/δ ~ N₀ (verified by (WIN)):

  ∫_{t₀}^{t₁} |Σ_{n≤N₀} a_n^{(k)} e^{−it·log n}|² dt
    = T · Σ_{n≤N₀} n^{−1} · (log n/|ϑ'(t₀)|)^{2k} · (1 + O(t₀^{−1/2})).
                                                                  ... (MV)

By (BOUND), each factor (log n/|ϑ'(t₀)|)^{2k} ≤ (1 + ε(t₀))^{2k}. The
|ζ|^{2k} averaging in (CV), combined with the mean-value bound
∫_{t₀}^{t₁} |ζ|^{2k} dt and (MV), gives:

  ‖F^{(k)}‖² ≤ (1 + ε(t₀))^{2k} · ‖F‖² · (1 + o(1)),          ... (FBD)

  ‖H^{(k)}‖² ≤ (2 + ε(t₀))^{2k} · ‖H‖² · (1 + o(1)),          ... (GBD)

  ‖E^{(k)}‖  = o(1) · ‖Y‖.                                      ... (EBD)

(EBD) follows from (Rbd) and (SV): |E^{(k)}(τ)| ≤ C_k · t₀^{−1/4−k/2} · B(τ).

By the triangle inequality:

  ‖Y^{(k)}‖_{L²([τ₀,2τ₀])}
    ≤ ‖F^{(k)}‖ + ‖H^{(k)}‖ + ‖E^{(k)}‖
    ≤ (2 + ε(t₀))^k · (1 + o(1)) · ‖Y‖_{L²([τ₀,2τ₀])}.         ... (DB)

Taking k-th roots:

  limsup_{k→∞} ‖Y^{(k)}‖^{1/k}_{L²([τ₀,2τ₀])} ≤ 2 + ε(t₀).   ... (SR)  □


───────────────────────────────────────────────────────────────────────────────
STEP 6. supp(Ŷ) ⊆ [−2, 0]
───────────────────────────────────────────────────────────────────────────────

PART A: supp(Ŷ) ⊆ [−2, 2].

For any ε > 0, (SR) holds with bound 2 + ε for all sufficiently large τ₀.
Andersen–de Jeu (Trans. Amer. Math. Soc. 362 (2010), Theorem 2.7) applied
along a sequence τ_j → ∞ satisfying (SR) with bound 2 + ε gives

  supp(Ŷ) ⊆ [−(2+ε), 2+ε].

Since ε > 0 is arbitrary: supp(Ŷ) ⊆ [−2, 2].

PART B: supp(Ŷ) ∩ (0, ∞) = ∅.

We establish H[Y] = −iY in L²_loc, which is equivalent to Ŷ(ξ) = 0 for
all ξ > 0. (The Hilbert transform acts as multiplication by −i·sgn(ξ) in the
Fourier domain, so H[Y]^(ξ) = −i·sgn(ξ)·Ŷ(ξ); H[Y] = −iY forces
−i·sgn(ξ)·Ŷ(ξ) = −i·Ŷ(ξ), i.e., Ŷ(ξ)·(1 − sgn(ξ)) = 0, hence Ŷ(ξ) = 0
for all ξ > 0.)

BEDROSIAN CONDITION. Each term in (★) is a product B(τ) · e^{iφ_n(τ)} where:

  — B(τ) is slowly varying with bandwidth O(τ₀^{−1+ε}) on [τ₀, 2τ₀] (Step 4).
  — φ_n'(τ) = ω_n(τ) ≤ 0 everywhere (Step 3), so the carrier is
    analytic-signal-like with non-positive instantaneous frequency.

Near zeros of ζ, B(τ) = [|ϑ'(t)| · |ζ|²]^{−1/2} blows up, but the product
B(τ) · S_j(σ(τ)) = (part of Y) · |ζ| remains bounded, since
|Y(τ)| = 1/√|ϑ'(σ(τ))| carries no |ζ| factor. By the cancellation
established in (L2), each term B · S_j and B · R lies in L²_loc, so the
blowup of B near zeros is always cancelled by the corresponding zero of the
RS sum. The Bedrosian theorem therefore applies to Y as a whole in the
averaged L²_loc sense: the amplitude |Y(τ)| = 1/√|ϑ'(σ(τ))| is smooth and
slowly varying (bandwidth O(τ₀^{−1+ε}) by (SV)), and all phase components
of Y have non-positive instantaneous frequency throughout (★). For all
sufficiently large τ₀ the bandwidth of |Y| is strictly less than the
minimum magnitude of the non-zero carrier frequencies (which are O(1/|ζ|²)
on average, bounded away from zero in L² mean by the mean-value theorem),
so the Bedrosian condition is satisfied in L²_loc and

  H[Y] = −iY   in L²_loc.

CONSEQUENCE. H[Y] = −iY implies Ŷ(ξ) = 0 for all ξ > 0, so

  supp(Ŷ) ∩ (0, ∞) = ∅.

CONCLUSION. Combining Parts A and B:

  supp(Ŷ) ⊆ [−2, 0].                                                       ■


═══════════════════════════════════════════════════════════════════════════════
END OF PROOF
