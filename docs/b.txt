§9.10. The Fractional Fokas–Its–Kitaev Theorem

The meromorphy of the transformed solution g(z) = y(z^{1/μ}), established in
Proposition 9.21 via the matrix Mittag-Leffler function, permits an exact
Riemann–Hilbert characterization of the Padé approximant that does not rely
on the abstract Stahl compact of §3. The contour, the weight, and the jump
matrix are all determined explicitly by the zeros of the Mittag-Leffler
entries.

§9.10.1. The discrete spectral measure

Let W(z) = E_{μ,1}(Az) be the matrix Mittag-Leffler function with

    A = (0 1; -c_0 c_2 c_1).

Write:

    W(z) = (E_{11}(z) E_{12}(z); E_{21}(z) E_{22}(z))

where each

    E_{ij}(z) = Σ_{k=0}^{∞} (A^k)_{ij} / Γ(kμ+1) z^k

is an entire function.

By Proposition 9.21,

    g(z) = -E_{21}(z)/(c_2 E_{11}(z))

is meromorphic on ℂ with discrete pole set:

    P_g = {p ∈ ℂ : E_{11}(p) = 0}

The poles are simple for generic parameters (c_0, c_1, c_2). At each pole
p ∈ P_g:

    Res(g, p) = -E_{21}(p) / (c_2 E_{11}'(p))

Definition 9.32 (Mittag-Leffler spectral measure). The discrete measure on
ℂ defined by:

    μ_g = Σ_{p ∈ P_g} Res(g, p) · δ_p

is the spectral measure of the fractional Riccati equation. Its total mass
is:

    μ_g(ℂ) = Σ_{p ∈ P_g} Res(g, p) = -a_1 = -c_0 / Γ(μ+1)

and its moments are exactly the Müntz coefficients:

    a_k = ∫_ℂ s^{k-1} dμ_g(s) = Σ_{p ∈ P_g} Res(g, p) · p^{k-1},  k ≥ 1

§9.10.2. The Riemann–Hilbert problem with explicit data

Theorem 9.33 (Fractional Fokas–Its–Kitaev representation). Let
g(z) = Σ_{k=1}^{∞} a_k z^k be the meromorphic function of Proposition
9.21, with spectral measure μ_g. Let n ≥ 1 and assume
τ_k = det H^{(k)} ≠ 0 for k = 1, …, n, where
H^{(k)} = (a_{i+j})_{0 ≤ i,j ≤ k-1}.

Then there exists a unique 2 × 2 matrix-valued function
Y: ℂ \ P_g → GL_2(ℂ) satisfying:

(i) Analyticity. Y is analytic on ℂ \ P_g.

(ii) Jump condition. For every p ∈ P_g:

    lim_{z → p} Y(z) has a simple pole with residue

    Res(Y, p) = (0  2πi Q_n(p) Res(g,p);
                 0  2πi Q̃_{n-1}(p) Res(g,p))

Equivalently, for any contour Σ passing through a finite subset
P_g^{(N)} ⊂ P_g:

    Y_+(s) = Y_-(s) (1  2πi w_N(s); 0  1),  s ∈ Σ

where

    w_N(s) = Σ_{p ∈ P_g^{(N)}} Res(g,p) · δ_p(s).

(iii) Asymptotics at infinity.

    Y(z) = (I + O(z^{-1})) (z^n  0; 0  z^{-n}),  z → ∞

(iv) Entry recovery.

    Y_{11}(z) = Q_n(z),
    Y_{12}(z) = Σ_{p ∈ P_g} Q_n(p) Res(g,p) / (p - z)

where Q_n is the monic orthogonal polynomial of degree n for μ_g.

Proof. The existence and uniqueness follow from the general
Fokas–Its–Kitaev theory (Theorem 2.3) once the moment functional is
specified. The moment functional L[p] = ∫ p(s) dμ_g(s) generates the
Hankel determinants τ_n by construction. The discrete support of μ_g
implies that the Cauchy transform reduces to a sum over poles, and the
jump across any contour through P_g picks up the residue contributions
2πi · Res(g,p) at each pole. The asymptotic condition follows from the
degree-n polynomial growth of Q_n and the O(z^{-1}) decay of the
Cauchy transform. ∎

§9.10.3. The Padé approximant as resolvent and as Riemann–Hilbert ratio

Corollary 9.34 (Three representations of the Padé approximant). Under
the hypotheses of Theorem 9.33, the diagonal Padé approximant [n/n]_g
admits three equivalent representations:

(a) Rational form:

    [n/n]_g(z) = P_n(z) / Q_n(z)

(b) Resolvent form:

    [n/n]_g(z) = ⟨e_0, (J_n - zI)^{-1} e_0⟩
               = Σ_{j=1}^n |⟨e_0, ψ_j^{(n)}⟩|^2 / (λ_j^{(n)} - z)

where J_n is the n × n Jacobi matrix with spectral data
(λ_j^{(n)}, ψ_j^{(n)}).

(c) Riemann–Hilbert form:

    [n/n]_g(z) = Y_{12}(z) / Y_{11}(z)
               = [Σ_{p ∈ P_g} Q_n(p) Res(g,p) / (p - z)] / Q_n(z)

Proof. (a) is the definition. (b) is Theorem 6.4. (c) follows from
Theorem 9.33(iv) and the identity:

    Σ_{p ∈ P_g} Q_n(p) Res(g,p) / (p - z) = P_n(z)

which holds because the left side is the Cauchy transform of the discrete
measure Q_n · μ_g, and the orthogonal polynomial property ensures that
the first n moments of the remainder vanish. ∎

§9.10.4. The Stahl compact for the Mittag-Leffler meromorphic function

Theorem 9.35 (Stahl compact of the fractional Riccati solution). Let
g(z) = -E_{21}(z)/(c_2 E_{11}(z)) be the meromorphic continuation of the
transformed fractional Riccati solution. Then:

(i) g ∈ S (the Stahl class).

(ii) The Stahl compact Δ_g satisfies:

    Δ_g = P̄_g ∪ γ

where γ is a finite union of arcs connecting the poles of g to
infinity, selected by the S-property.

(iii) The equilibrium measure μ_{Δ_g} is the weak-* limit of the
normalized counting measures of the Padé poles:

    (1/n) Σ_{j=1}^n δ_{λ_j^{(n)}} ⇀ μ_{Δ_g}  as n → ∞

(iv) The Green's function g_{Δ_g}(z, ∞) with pole at infinity satisfies:

    lim sup_{n → ∞} |g(z) - [n/n]_g(z)|^{1/n}
        ≤ e^{-g_{Δ_g}(z,∞)},  z ∈ ℂ \ Δ_g

Proof. (i) Meromorphic functions on ℂ with discrete pole set belong to
S because the pole set has logarithmic capacity zero. (ii) The Stahl
compact for a meromorphic function is the union of the pole set and the
arcs connecting poles to infinity that minimize capacity; the S-property
selects these arcs. (iii) This is Theorem 7.3 applied to the orthogonal
polynomial zeros, which coincide with the eigenvalues of J_n (Corollary
6.5). (iv) This is Corollary 3.8. ∎

§9.10.5. The self-consistency theorem

Theorem 9.36 (Closure of the Mittag-Leffler–Padé–Riemann-Hilbert loop).
The following diagram commutes:

    Fractional Riccati
          |
          | Cole-Hopf
          v
    Matrix Mittag-Leffler
          |
          | z = t^μ
          v
    Meromorphic g(z)
          |
          | moments
          v
    Spectral measure μ_g
          |
          | Hankel
          v
    Jacobi operator J_n
          |
          | resolvent
          v
    Padé approximant [n/n]_g
          |
          | Y_{12}/Y_{11}
          v
    Riemann-Hilbert Y
          |
          | rational function
          v
    Stieltjes transform of μ_g|_{P_g^{(n)}}

Moreover, as n → ∞:

- The Padé poles λ_j^{(n)} condense on Δ_g (Theorem 9.35).
- The Padé approximant [n/n]_g(z) → g(z) uniformly on compact subsets
  of [0, ∞) (Theorem 9.24).
- The Riemann-Hilbert matrix Y converges to the exact factorization for
  the infinite pole set P_g.

Proof. The commutativity of the upper triangle follows from Proposition
9.21: the Mittag-Leffler solution yields the meromorphic function g,
whose Taylor coefficients are the moments of μ_g. The left triangle is
Theorem 6.4 (Jacobi resolvent equals Padé). The right triangle is
Corollary 9.34(c) (Riemann–Hilbert ratio equals Padé). The convergence
statements are Theorems 9.24 and 9.35. ∎

§9.10.6. Theoretical significance

The fractional Fokas–Its–Kitaev theorem (Theorem 9.33) achieves what the
abstract theory of §§2–3 could not: it identifies the explicit data of
the Riemann–Hilbert problem for a concrete meromorphic function arising
from fractional calculus. The weight w(s) is not an arbitrary density but
a sum of Dirac masses at the zeros of the Mittag-Leffler entry E_{11}(z),
with masses given by the residues of the ratio
-E_{21}(z)/(c_2 E_{11}(z)). The contour Σ is not selected by an abstract
variational principle but is the Stahl compact of a globally meromorphic
function, which decomposes into the pole set plus connecting arcs.

This explicit identification is possible because the Cole–Hopf
linearization reduces the nonlinear fractional Riccati equation to a
linear system solvable by special functions. The resulting meromorphic
structure makes the fractional Riccati a test case for the general
Riemann–Hilbert–Padé theory: every object (contour, weight, orthogonal
polynomials, Jacobi operator, equilibrium measure) can in principle be
computed from the Mittag-Leffler series, even though the pole locations
are transcendental numbers.

The closure theorem (Theorem 9.36) guarantees that the three
paradigms—special-function exact solution, rational Padé approximation,
and matrix Riemann–Hilbert factorization—are mathematically equivalent for
the fractional Riccati equation, with explicit convergence certificates
and computable error bounds.
