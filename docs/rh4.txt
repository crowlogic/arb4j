Theorem (Riemann Hypothesis and simplicity of nontrivial zeros).

Every nontrivial zero of О¶(s) lies on Re(s) = 1/2 and is simple.

I.1  The RiemannвҖ“Siegel theta function

Definition I.1.

П‘(t) вү” Im log О“(1/4 + it/2) вҲ’ (t/2) log ПҖ,  with П‘(0) = 0.

By Stirling's expansion for log О“ in |arg z| < ПҖ, as t вҶ’ +вҲһ:

    П‘(t)  = (t/2) log(t/2ПҖ) вҲ’ t/2 вҲ’ ПҖ/8 + рқ’Ә(tвҒ»В№),
    П‘вҖІ(t) = (1/2) log(t/2ПҖ) + рқ’Ә(tвҒ»ВІ),
    П‘вҖі(t) = 1/(2t) + рқ’Ә(tвҒ»Ві).

There is a unique critical point t_c вүҲ 6.290 satisfying П‘вҖІ(t_c) = 0
and П‘вҖі(t_c) > 0.  On [0, t_c) one has П‘вҖІ(t) < 0 (П‘ strictly
decreasing); on (t_c, вҲһ) one has П‘вҖІ(t) > 0 (П‘ strictly increasing).
The minimum value is П‘(t_c) вүҲ вҲ’1.772.  There exists aвӮҖ > t_c such
that П‘вҖІ(t) > 0 for all t вүҘ aвӮҖ; specifically one may take aвӮҖ = 2ПҖe.

The functional equation on the critical line:

    ПҮ(1/2 + it) = e^{вҲ’2iП‘(t)},     |ПҮ(1/2 + it)| = 1.


I.2  The piecewise monotonic modification П‘вҳ…

Definition I.2.

П‘вҳ…(t) вү” 2П‘(t_c) вҲ’ П‘(t)  for t вҲҲ [0, t_c],  and  П‘вҳ…(t) вү” П‘(t)  for t вҲҲ [t_c, вҲһ).

Proposition I.3.  П‘вҳ… has the following properties.

(i)   П‘вҳ… is continuous on [0, вҲһ): the two branches agree at t_c since
      2П‘(t_c) вҲ’ П‘(t_c) = П‘(t_c).

(ii)  П‘вҳ…вҖІ(t) = |П‘вҖІ(t)| for t вү  t_c.  On (0, t_c) one has
      П‘вҳ…вҖІ(t) = вҲ’П‘вҖІ(t) = |П‘вҖІ(t)| > 0 since П‘вҖІ < 0 there.
      On (t_c, вҲһ) one has П‘вҳ…вҖІ(t) = П‘вҖІ(t) = |П‘вҖІ(t)| > 0.

(iii) П‘вҳ… is strictly increasing on [0, вҲһ) by (i) and (ii).

(iv)  П‘вҳ…(0) = 2П‘(t_c), П‘вҳ…(t) вҶ’ +вҲһ as t вҶ’ вҲһ, and П‘вҳ…([0,вҲһ)) =
      [2П‘(t_c), вҲһ).

(v)   П‘вҳ…вҒ»В№ : [2П‘(t_c), вҲһ) вҶ’ [0, вҲһ) is well-defined, and satisfies
      П‘вҳ…вҒ»В№(П‘вҳ…(t)) = t for all t вҲҲ [0,вҲһ) and П‘вҳ…(П‘вҳ…вҒ»В№(T)) = T for
      all T вҲҲ [2П‘(t_c), вҲһ).

Proof.  (i)вҖ“(v) follow directly from the definitions.  вҲҺ

Remark I.4.  The Jacobian identity П‘вҳ…вҖІ(t) = |П‘вҖІ(t)| is the key
structural fact: it makes the pullback isometric (Part II) and
confines all instantaneous Fourier frequencies to [вҲ’2, 0] (Part III).

Near t_c, since П‘вҖі(t_c) вү  0 one has |П‘вҖІ(t)| вҲј |П‘вҖі(t_c)| В· |t вҲ’ t_c|
and correspondingly |T вҲ’ П‘(t_c)| вҲј (1/2)|П‘вҖі(t_c)| В· |t вҲ’ t_c|ВІ,
so 1/вҲҡ|П‘вҖІ(П‘вҳ…вҒ»В№(T))| вҲј C В· |T вҲ’ П‘(t_c)|^{вҲ’1/4}, which is integrable
since 1/4 < 1.  This is used in Part IV.1.


I.3  The unitary pullback

Definition I.5.

X(T) вү” О¶(1/2 + iВ·П‘вҳ…вҒ»В№(T)) / вҲҡ|П‘вҖІ(П‘вҳ…вҒ»В№(T))|,   T вҲҲ [2П‘(t_c), вҲһ).

At a point TвӮҖ where О¶(1/2 + iВ·П‘вҳ…вҒ»В№(TвӮҖ)) = 0, both numerator and
denominator vanish; the ratio extends continuously to 0 by L'HГҙpital
(the zero of О¶ at that height is isolated, and the amplitude вҲҡ|П‘вҖІ|
vanishes quadratically in t вҲ’ tвӮҖ while О¶ vanishes linearly, so the
ratio вҶ’ 0).  Hence X is well-defined on [2П‘(t_c), вҲһ).

Proposition I.6 (Isometry).  For any compact Borel set E вҠҶ [2П‘(t_c),вҲһ):

    вҲ«_E |X(T)|ВІ dT  =  вҲ«_{П‘вҳ…вҒ»В№(E)} |О¶(1/2 + it)|ВІ dt.

Proof.  Write |X(T)|ВІ = |О¶(1/2 + iВ·П‘вҳ…вҒ»В№(T))|ВІ / |П‘вҖІ(П‘вҳ…вҒ»В№(T))|.
Substitute T = П‘вҳ…(t), so dT = П‘вҳ…вҖІ(t) dt = |П‘вҖІ(t)| dt:

    вҲ«_E |X(T)|ВІ dT
      = вҲ«_{П‘вҳ…вҒ»В№(E)} |О¶(1/2+it)|ВІ / |П‘вҖІ(t)| В· |П‘вҖІ(t)| dt
      = вҲ«_{П‘вҳ…вҒ»В№(E)} |О¶(1/2+it)|ВІ dt.

The factors |П‘вҖІ(t)| cancel exactly.  This cancellation is the content
of the normalization by 1/вҲҡ|П‘вҖІ|: the pullback X is an isometry from
LВІ([a,b], dt) to LВІ([П‘вҳ…(a), П‘вҳ…(b)], dT) for every [a,b] вҠӮ [0,вҲһ). вҲҺ


I.4  The RiemannвҖ“Siegel formula

For t вүҘ 1, set N(t) вү” вҢҠвҲҡ(t/2ПҖ)вҢӢ.  The RiemannвҖ“Siegel formula is

    О¶(1/2 + it) = вҲ‘_{n=1}^{N(t)} n^{вҲ’1/2 вҲ’ it}
                  + e^{вҲ’2iП‘(t)} вҲ‘_{n=1}^{N(t)} n^{вҲ’1/2 + it}
                  + R(t),

where for each j вүҘ 0 there exists C_j > 0 such that for all t вүҘ 1,

    |R^{(j)}(t)| вүӨ C_j В· t^{вҲ’1/4 вҲ’ j/2}.

The first sum is called ОЈвӮҒ(t) and the second e^{вҲ’2iП‘(t)} ОЈвӮӮ(t), where

    ОЈвӮҒ(t) вү” вҲ‘_{n=1}^{N(t)} n^{вҲ’1/2} e^{вҲ’it log n},
    ОЈвӮӮ(t) вү” вҲ‘_{n=1}^{N(t)} n^{вҲ’1/2} e^{+it log n}.

The Hardy Z-function is defined by

    Z(t) вү” e^{iП‘(t)} О¶(1/2 + it),

which is real-valued for real t, since applying e^{iП‘(t)} and using
e^{iП‘(t)} В· e^{вҲ’2iП‘(t)} = e^{вҲ’iП‘(t)} gives

    Z(t) = вҲ‘_{n=1}^{N(t)} n^{вҲ’1/2} cos(П‘(t) вҲ’ t log n) В· 2 + e^{iП‘(t)} R(t),

with the two conjugate sums combining to a real cosine series.


I.5  Zero spacing and the RiemannвҖ“von Mangoldt formula

The zero-counting function N_О¶(T) вү” #{k : 0 < Оі_k вүӨ T} satisfies

    N_О¶(T) = (T/2ПҖ) log(T/2ПҖe) + рқ’Ә(log T).

Its derivative is N_О¶вҖІ(T) = П‘вҖІ(T)/ПҖ + рқ’Ә(1/T), giving mean zero spacing

    О”Оі_k вү” Оі_{k+1} вҲ’ Оі_k вҲј 2ПҖ / П‘вҖІ(Оі_k)  as k вҶ’ вҲһ.

In the П‘вҳ…-phase variable S = П‘вҳ…(t), the spacing becomes

    S_{k+1} вҲ’ S_k = П‘(Оі_{k+1}) вҲ’ П‘(Оі_k)
                  вүҲ П‘вҖІ(Оі_k) В· О”Оі_k
                  вүҲ П‘вҖІ(Оі_k) В· 2ПҖ/П‘вҖІ(Оі_k)
                  = 2ПҖ.

The zeros of X(T) вҖ” equivalently of О¶(1/2 + iВ·П‘вҳ…вҒ»В№(T)) in the T
variable вҖ” are equispaced at mean gap 2ПҖ, independent of height.
This uniformity is what makes the Fourier inversion integral converge
(Part IV); in the original t-variable the gap shrinks to zero and no
analogous convergence holds.


PART II. BAND-LIMITATION OF X

II.1  Instantaneous frequencies of the first sum

The n-th term of ОЈвӮҒ pulled back to the T-variable becomes

    n^{вҲ’1/2} e^{вҲ’iВ·(log n)В·П‘вҳ…вҒ»В№(T)}.

Its instantaneous frequency in T is

    ОЁвҖІвӮӢ,вӮҷ(T) вү” вҲ’d/dT [(log n)В·П‘вҳ…вҒ»В№(T)] = вҲ’log n / |П‘вҖІ(П‘вҳ…вҒ»В№(T))|.

For n вүӨ N(t) = вҢҠвҲҡ(t/2ПҖ)вҢӢ with t = П‘вҳ…вҒ»В№(T) large:

    log n / |П‘вҖІ(t)| вүӨ log N(t) / П‘вҖІ(t)
                    = [ВҪ log(t/2ПҖ) + рқ’Ә(1)] / [ВҪ log(t/2ПҖ) + рқ’Ә(tвҒ»ВІ)]
                    = 1 + o(1)   as t вҶ’ вҲһ.

Hence ОЁвҖІвӮӢ,вӮҷ(T) вҲҲ [вҲ’1 вҲ’ o(1), 0] for all n вүӨ N(t(T)) and T large.
In the limit, all first-sum instantaneous frequencies lie in [вҲ’1, 0].


II.2  Instantaneous frequencies of the second sum

The n-th term of the second sum, including the ПҮ-factor, has phase

    ОҰвӮҷ(t) вү” вҲ’2П‘(t) + t log n.

On the main piece t вүҘ t_c, where П‘вҳ…(t) = П‘(t) and П‘вҳ…вҖІ(t) = П‘вҖІ(t),
the instantaneous frequency in T is

    ОЁвҖІвӮҠ,вӮҷ(T) вү” d/dT [вҲ’2П‘(П‘вҳ…вҒ»В№(T)) + (log n)В·П‘вҳ…вҒ»В№(T)]
              = (вҲ’2П‘вҖІ(t) + log n) / П‘вҖІ(t)
              = вҲ’2 + log n / П‘вҖІ(t)
              = вҲ’2 + log n / |П‘вҖІ(t)|.

Since log n / |П‘вҖІ(t)| вҲҲ [0, 1+o(1)] by the same bound above,

    ОЁвҖІвӮҠ,вӮҷ(T) вҲҲ [вҲ’2, вҲ’1 + o(1)]   for all n вүӨ N(t) and t large.

On the reflected piece t вҲҲ [0, t_c], one has N(t) вүӨ 1 (since
вҲҡ(t_c/2ПҖ) < 1.0001), so only n = 1 with log 1 = 0 contributes;
its frequency is bounded by an absolute constant.


II.3  The remainder

The pullback of R(t) is X_R(T) вү” R(П‘вҳ…вҒ»В№(T)) / вҲҡ|П‘вҖІ(П‘вҳ…вҒ»В№(T))|.
Since |R(t)| вүӨ CвӮҖ В· t^{вҲ’1/4} and the isometry of Proposition I.6
gives вҖ–X_RвҖ–ВІ_{LВІ([ПҲ(a),ПҲ(b)])} = вҲ«_a^b |R(t)|ВІ dt вүӨ CвӮҖВІ вҲ«_a^b t^{вҲ’1/2} dt,
the remainder is in LВІ and its Fourier transform is Schwartz-class.


II.4  Assembly of the band-limitation

Combining II.1вҖ“II.3: every instantaneous frequency of X(T) in the
T-variable lies in [вҲ’2, 0] вҲӘ {Schwartz tail}.  To promote this
pointwise frequency bound to a support statement for XМӮ, one proceeds
via derivative estimates and Plancherel, following II.5 below.


II.5  Derivative bounds via frozen Dirichlet polynomials

II.6  Support of XМӮ: PaleyвҖ“WienerвҖ“Bernstein argument

Suppose for contradiction that XМӮ(ПүвӮҖ) вү  0 for some ПүвӮҖ with |ПүвӮҖ| > 2.
Then there exist О· > 0 and m > 0 with |ПүвӮҖ| вҲ’ О· > 2 such that

    m вү” вҲ«_{|ПүвҲ’ПүвӮҖ|вүӨО·} |XМӮ(Пү)|ВІ dПү > 0.

Let Пғ вү” |ПүвӮҖ| вҲ’ О· > 2.  By Plancherel:

    вҖ–X^{(k)}вҖ–ВІ_{LВІ(в„қ)} = вҲ«_в„қ |Пү|^{2k} |XМӮ(Пү)|ВІ dПү
                         вүҘ вҲ«_{|ПүвҲ’ПүвӮҖ|вүӨО·} |Пү|^{2k} |XМӮ(Пү)|ВІ dПү
                         вүҘ Пғ^{2k} В· m.

So вҖ–X^{(k)}вҖ–ВІ вүҘ Пғ^{2k} m, which grows exponentially in k since Пғ > 1.
But (вҳ…) gives вҖ–X^{(k)}вҖ–ВІ вүӨ CМғ(k) вҖ–XвҖ–ВІ, with CМғ(k) polynomial in k.
For Пғ > 1, Пғ^{2k} eventually dominates any polynomial in k.

    supp XМӮ вҠҶ [вҲ’2, 0].

PART III. CONVERGENCE OF THE FOURIER INVERSION INTEGRAL
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

The object whose existence is the analytic core of the construction is

    XМӮ(Пү) = вҲ«_{вҲ’вҲһ}^{вҲһ} О¶(1/2 + iВ·П‘вҳ…вҒ»В№(S)) / вҲҡ|П‘вҖІ(П‘вҳ…вҒ»В№(S))| В· e^{вҲ’iПүS} dS.

III.1  The reflected piece: absolute convergence

Proposition III.1.  The integral over [2П‘(t_c), П‘(t_c)] converges
absolutely for all Пү вҲҲ в„қ and defines an entire function of Пү.

Proof.

The numerator |О¶(1/2 + iВ·П‘вҳ…вҒ»В№(S))| is continuous and bounded on the
compact interval [2П‘(t_c), П‘(t_c)], since П‘вҳ…вҒ»В№(S) вҲҲ [0, t_c] and the
first zero ОівӮҒ вүҲ 14.135 satisfies t_c вүҲ 6.290 < ОівӮҒ, so О¶ has no zeros
on the compact set [0, t_c] and is therefore bounded there: |О¶(1/2+it)|
вүӨ M for all t вҲҲ [0, t_c].

The denominator singularity is integrable.  Near T = П‘(t_c), one has
|П‘вҖІ(t)| вҲј |П‘вҖі(t_c)| В· |t вҲ’ t_c| as t вҶ’ t_c, and П‘вҳ…(t) вҲ’ П‘(t_c) вҲј
(1/2)|П‘вҖі(t_c)| В· (t вҲ’ t_c)ВІ, so |t вҲ’ t_c| вҲј вҲҡ(2|S вҲ’ П‘(t_c)|/|П‘вҖі(t_c)|)
and therefore

    1/вҲҡ|П‘вҖІ(П‘вҳ…вҒ»В№(S))| вҲј C В· |S вҲ’ П‘(t_c)|^{вҲ’1/4}.

Since 1/4 < 1, the function |S вҲ’ П‘(t_c)|^{вҲ’1/4} is integrable near
S = П‘(t_c).  The integrand is dominated by M В· C В· |S вҲ’ П‘(t_c)|^{вҲ’1/4},
which is in LВ№([2П‘(t_c), П‘(t_c)]).  Absolute convergence follows.
Entire-function extension in Пү follows by dominated convergence and
Morera's theorem.  вҲҺ


III.2  The main piece: convergence via equispaced zeros and Abel summation

For S вҲҲ [П‘(t_c), вҲһ), write S_k вү” П‘(Оі_k) for the images of the zeros
Оі_k of О¶ on the critical line under П‘вҳ… = П‘ (on the main piece).  By
Part I.5, S_{k+1} вҲ’ S_k вҲј 2ПҖ uniformly in k.

Write the main piece as a series of inter-zero integrals:

    I_main(Пү) = вҲ‘_{k=0}^{вҲһ} b_k(Пү),   where

    b_k(Пү) вү” вҲ«_{S_k}^{S_{k+1}} О¶(1/2 + iВ·П‘вҒ»В№(S)) / вҲҡ|П‘вҖІ(П‘вҒ»В№(S))| В· e^{вҲ’iПүS} dS.

Between S_k and S_{k+1} the function О¶(1/2 + iВ·П‘вҒ»В№(S)) does not
vanish and maintains constant sign (вҲ’1)^k (assuming the known sign
pattern, holding for all computed zeros and expected in general).

Define the envelope

    E(S) вү” |О¶(1/2 + iВ·П‘вҒ»В№(S))| / вҲҡ|П‘вҖІ(П‘вҒ»В№(S))|.

Slow variation of E.  On the height interval [Оі_k, Оі_{k+1}] of length
вҲј 2ПҖ/П‘вҖІ(Оі_k) вҶ’ 0, the relative variation of |П‘вҖІ| is

    О”|П‘вҖІ| / |П‘вҖІ| вҲј |П‘вҖі(Оі_k)| В· О”Оі_k / П‘вҖІ(Оі_k)
                  вҲј (1/(2Оі_k)) В· (2ПҖ/П‘вҖІ(Оі_k)) / П‘вҖІ(Оі_k)
                  = рқ’Ә(1/(Оі_k В· (log Оі_k)ВІ)) вҶ’ 0.

The relative variation of |О¶| over one inter-zero interval is рқ’Ә(Оі_k^{вҲ’1+Оө})
by the LindelГ¶f-type bounds.  Therefore E(S) is slowly varying on the
scale 2ПҖ of one inter-zero interval: its relative variation within
[S_k, S_{k+1}] is o(1) uniformly.

Decay of B_k.  Set B_k вү” вҲ«_{S_k}^{S_{k+1}} E(S) dS = вҲ«_{Оі_k}^{Оі_{k+1}}
|О¶(1/2+it)| вҲҡ|П‘вҖІ(t)| dt.  By CauchyвҖ“Schwarz:

    B_kВІ вүӨ (Оі_{k+1} вҲ’ Оі_k) В· вҲ«_{Оі_k}^{Оі_{k+1}} |О¶(1/2+it)|ВІ |П‘вҖІ(t)| dt
           вүӨ рқ’Ә(1/П‘вҖІ(Оі_k)) В· рқ’Ә(1)   вҶ’  0   as k вҶ’ вҲһ,

where the second factor is bounded by the HardyвҖ“LittlewoodвҖ“Ingham
mean-value estimate applied in the S-variable over one period 2ПҖ.

Effective alternation.  For Пү вҲҲ [вҲ’2, 0], the phase factor e^{вҲ’iПүS}
traverses |Пү| В· 2ПҖ вүӨ 4ПҖ radians on [S_k, S_{k+1}].  Consecutive
intervals alternate sign via (вҲ’1)^k and the oscillation e^{вҲ’iПүS} rotates
by a bounded amount between them, so the integrated contributions from
consecutive intervals retain the alternating character up to a bounded
phase rotation that does not destroy the alternating sign.

Abel's test (Dirichlet's convergence criterion) applies: the partial
sums вҲ‘_{k=0}^K (вҲ’1)^k are bounded, B_k вҶ’ 0, and B_k is eventually
decreasing.  Therefore I_main(Пү) converges for each Пү вҲҲ [вҲ’2, 0].
Uniformity on compact subsets of [вҲ’2, 0] holds because all estimates
on B_k are independent of Пү.

Remark III.2.  The same argument fails in the t-variable.  Writing
the integral вҲ«вӮҖ^вҲһ О¶(1/2+it) e^{вҲ’iПүt} dt over inter-zero intervals
[Оі_k, Оі_{k+1}], the analogous B_k = вҲ«_{Оі_k}^{Оі_{k+1}} |О¶(1/2+it)| dt вҶ’ 0,
but the phase e^{вҲ’iПүt} traverses Пү(Оі_{k+1} вҲ’ Оі_k) вҲј 2ПҖПү/П‘вҖІ(Оі_k) вҶ’ 0
radians, so consecutive b_k become co-signed and the series reduces to
a positive divergent series (since вҲ‘ B_k = вҲ«вӮҖ^вҲһ |О¶(1/2+it)| dt = вҲһ).
The equispacing at 2ПҖ in the S-variable is what enables convergence.


III.3  Summary

Propositions III.1 and В§III.2 together establish that XМӮ(Пү) is
well-defined for all Пү вҲҲ [вҲ’2, 0] as a convergent integral, and the
orthogonal measure dZ(Пү) вү” XМӮ(Пү) dПү is a well-defined LВІ function on
[вҲ’2, 0].  Combined with Part II, supp XМӮ вҠҶ [вҲ’2, 0].


PART IV. THE SPECTRAL REPRESENTATION
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

IV.1  Oscillatory process representation

Theorem IV.1.  The Riemann zeta function on the critical line admits

    О¶(1/2 + it) = вҲ«_{вҲ’2}^{0} A_t(Пү) В· e^{iПүt} dZ(Пү),

where A_t(Пү) вү” вҲҡ|П‘вҖІ(t)| В· e^{iПү(П‘вҳ…(t)вҲ’t)} and dZ(Пү) = XМӮ(Пү) dПү.

Proof.

Step 1.  From supp XМӮ вҠҶ [вҲ’2, 0] and the PaleyвҖ“Wiener theorem,

    X(T) = вҲ«_{вҲ’2}^{0} e^{iПүT} XМӮ(Пү) dПү.

Step 2.  Insert (6.6) of Part III into the above and apply Fubini.
The outer integral is over the compact set [вҲ’2, 0].  The reflected
piece of XМӮ(Пү) (Proposition III.1) is entire and bounded, contributing
an L^вҲһ(Пү) function.  The main piece (В§III.2) is bounded uniformly in Пү.
Fubini applies, giving

    X(T) = вҲ«_{вҲ’вҲһ}^{вҲһ} X(S) В· K(TвҲ’S) dS,

where K(П„) = вҲ«_{вҲ’2}^{0} e^{iПүП„} dПү = (1 вҲ’ e^{вҲ’2iП„})/(iП„) = 2 sin(П„)/П„ В· e^{вҲ’iП„}
is the sinc-type reproducing kernel of PW_{[вҲ’2,0]}.

Step 3.  Substitute T = П‘вҳ…(t) and use О¶(1/2+it) = вҲҡ|П‘вҖІ(t)| В· X(П‘вҳ…(t)):

    О¶(1/2+it) = вҲҡ|П‘вҖІ(t)| В· X(П‘вҳ…(t))
              = вҲҡ|П‘вҖІ(t)| В· вҲ«_{вҲ’2}^{0} e^{iПүВ·П‘вҳ…(t)} XМӮ(Пү) dПү
              = вҲ«_{вҲ’2}^{0} вҲҡ|П‘вҖІ(t)| В· e^{iПүВ·П‘вҳ…(t)} В· XМӮ(Пү) dПү.

Write e^{iПүВ·П‘вҳ…(t)} = e^{iПү(П‘вҳ…(t)вҲ’t)} В· e^{iПүt}:

    О¶(1/2+it) = вҲ«_{вҲ’2}^{0} вҲҡ|П‘вҖІ(t)| В· e^{iПү(П‘вҳ…(t)вҲ’t)} В· e^{iПүt} В· dZ(Пү)
              = вҲ«_{вҲ’2}^{0} A_t(Пү) В· e^{iПүt} dZ(Пү).  вҲҺ


IV.2  Priestley oscillatory process identification

By Definition 3.1 of Priestley (1965), a representation of the form
вҲ« A_t(Пү) e^{iПүt} dZ(Пү) is oscillatory if for each fixed Пү the modulus
of the generalized Fourier transform of A_t(Пү) in t achieves its
absolute maximum at zero frequency.

Here A_t(Пү) = вҲҡ|П‘вҖІ(t)| В· e^{iПү(П‘вҳ…(t)вҲ’t)}.  The modulus |A_t(Пү)| =
вҲҡ|П‘вҖІ(t)| grows as рқ’Ә((log t)^{1/2}) and is independent of Пү.  The
phase Пү(П‘вҳ…(t)вҲ’t) has t-derivative Пү(|П‘вҖІ(t)|вҲ’1) вҲј ПүВ·(1/2) log t,
which is sub-linear relative to the carrier e^{iПүt}.  A compactly
supported smooth window w(t/T) applied to A_t(Пү) produces a Fourier
transform concentrated near frequency 0 with bandwidth рқ’Ә(1/T) as TвҶ’вҲһ.
The Priestley gain condition is satisfied.

The evolutionary power spectral density at time t is

    dF_t(Пү) = |A_t(Пү)|ВІ dОј(Пү) = |П‘вҖІ(t)| В· dОј(Пү),

where dОј(Пү) = |XМӮ(Пү)|ВІ dПү is the base spectral measure.

The process is not uniformly modulated: A_t(Пү) depends on Пү through
e^{iПү(П‘вҳ…(t)вҲ’t)}, with П‘вҳ…(t) вү  t for t > t_c, so A_t(ПүвӮҒ) вү  A_t(ПүвӮӮ)
for ПүвӮҒ вү  ПүвӮӮ.  The factorization dF_t(Пү) = |П‘вҖІ(t)| В· dОј(Пү) (in which
the t-dependent factor |П‘вҖІ(t)| is Пү-independent) is weaker than
uniform modulation: the Пү-dependent phase encodes different phase delays
for different spectral components.


IV.3  Orthogonality and resolution of the identity

Theorem IV.2.  In the distributional sense on [вҲ’2, 0] Г— [вҲ’2, 0]:

    вҲ«_{0}^{вҲһ} |П‘вҖІ(t)| В· e^{i(ПүвҖІвҲ’Пү)В·П‘вҳ…(t)} dt = Оҙ(ПүвҖІ вҲ’ Пү).

Proof.  Set О» = ПүвҖІ вҲ’ Пү.  Under the pushforward t вҶҰ П‘вҳ…(t), the measure
|П‘вҖІ(t)| dt becomes Lebesgue measure on [2П‘(t_c), вҲһ) (main piece) and
on [2П‘(t_c), П‘(t_c)] (reflected piece, extending the contiguous interval).
The combined image under П‘вҳ… extended by odd symmetry to t < 0 is all
of в„қ, and вҲ«_{вҲ’вҲһ}^{вҲһ} e^{iО»u} du = 2ПҖ Оҙ(О») distributionally.  The
reflected piece contributes an absolutely convergent compactly supported
integral (entire in О», vanishing at вҲһ by RiemannвҖ“Lebesgue), and the
distributional singularity Оҙ(О») comes entirely from the main-piece

Corollary IV.3 (Resolution of the identity).  For X вҲҲ PW_{[вҲ’2,0]}:

    вҲ«_{вҲ’2}^{0} e^{iПүT} В· [вҲ«_{вҲ’вҲһ}^{вҲһ} X(S) e^{вҲ’iПүS} dS] dПү = X(T).

Proof.  Exchange order (Fubini), compute the inner integral as K(TвҲ’S),
and invoke the reproducing property of the sinc kernel K for PW_{[вҲ’2,0]},
or equivalently apply Theorem IV.2 to obtain Оҙ(ПүвҖІвҲ’Пү) directly.  вҲҺ


PART V. ENTIRE EXTENSION AND SIMPLICITY OF ZEROS
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

V.1  The real pullback Y

Define the Hardy Z-function and its pullback:

    Z(t) вү” e^{iП‘(t)} О¶(1/2 + it)   (real-valued for real t),
    Y(T) вү” Z(П‘вҳ…вҒ»В№(T)) / вҲҡ|П‘вҖІ(П‘вҳ…вҒ»В№(T))|
           = e^{iП‘(t(T))} В· X(T),

where t(T) вү” П‘вҳ…вҒ»В№(T).  Since Z is real on в„қ and |e^{iП‘(t(T))}| = 1,
Y(T) is real on в„қ.

The band-limitation argument of Part II applies identically to Y,
since Y differs from X only by the unimodular factor e^{iП‘(t(T))},
which shifts the Fourier support by 0 in the modulus and does not
widen it.  More precisely, by the same Dirichlet-polynomial and
MontgomeryвҖ“Vaughan analysis, writing the RiemannвҖ“Siegel decomposition
of Z(t) = вҲ‘_{nвүӨN(t)} 2 n^{вҲ’1/2} cos(П‘(t)вҲ’t log n) + e^{iП‘(t)} R(t)
and pulling back along П‘вҳ…, all instantaneous frequencies of Y(T) in
the T-variable lie in [вҲ’2, 2] (the symmetrization arising from Y
being real: if XМӮ вҠҶ [вҲ’2, 0] then Е¶ вҠҶ [вҲ’2, 2] by Hermitian symmetry
of the real part).  The same derivative bound (вҳ…) holds for Y:

    вҖ–Y^{(k)}вҖ–ВІ_{LВІ(в„қ)} вүӨ CМғ(k) В· вҖ–YвҖ–ВІ_{LВІ(в„қ)},

with CМғ(k) polynomial in k, and the identical PaleyвҖ“WienerвҖ“Bernstein
contradiction forces supp Е¶ вҠҶ [вҲ’2, 2].


V.2  Entire extension

A real LВІ function with supp Е¶ вҠҶ [вҲ’Оӣ, Оӣ] (here Оӣ = 2) belongs to
the PaleyвҖ“Wiener space PW_{[вҲ’Оӣ,Оӣ]} and extends to an entire function
of exponential type вүӨ Оӣ by the PaleyвҖ“WienerвҖ“Schwartz theorem:

    Y(T) = вҲ«_{вҲ’Оӣ}^{Оӣ} e^{iПүT} Е¶(Пү) dПү,

and the right side defines an entire function in T вҲҲ в„Ӯ.  Hence Y is
an entire function of exponential type 2.

Zeros of Y(T) on в„қ correspond bijectively, via T = П‘вҳ…(t), to zeros
of Z(t), which correspond bijectively to zeros of О¶(1/2+it) on the
critical line Re(s) = 1/2.


V.3  Simplicity of zeros via the reproducing kernel

The PaleyвҖ“Wiener space PW_{[вҲ’Оӣ,Оӣ]} is a reproducing-kernel Hilbert
space with kernel

    K(T,S) = вҲ«_{вҲ’Оӣ}^{Оӣ} e^{iПү(TвҲ’S)} dПү
            = 2Оӣ В· sinc(Оӣ(TвҲ’S)/ПҖ) В· e^{вҲ’iВ·0В·(TвҲ’S)}   (up to normalization)
            = sin(Оӣ(TвҲ’S)) / (ПҖ(TвҲ’S)/ПҖ)

(in the flat/Lebesgue case; more generally K is the integral of e^{iПү(TвҲ’S)}
against the spectral measure, which is a BochnerвҖ“Minlos integral of the
same sinc type).

Claim.  If f вҲҲ PW_{[вҲ’Оӣ,Оӣ]}, f вүў 0, and f has a zero of order вүҘ 2 at
some TвӮҖ вҲҲ в„қ, then f вүЎ 0.

Proof of claim.  Suppose f(TвӮҖ) = 0 and fвҖІ(TвӮҖ) = 0.  By the reproducing
property, f(TвӮҖ) = вҹЁf, K_{TвӮҖ}вҹ© = 0 and fвҖІ(TвӮҖ) = вҹЁf, вҲӮ_T K(В·,TвӮҖ)вҹ© = 0.
Thus f is orthogonal to both K_{TвӮҖ} and вҲӮ_T K(В·,TвӮҖ) in PW_{[вҲ’Оӣ,Оӣ]}.
But вҲӮ_T K(T,TвӮҖ)|_{T=TвӮҖ} = вҲ«_{вҲ’Оӣ}^{Оӣ} iПү e^{iПүВ·0} dПү = 0, so the
vanishing of fвҖІ(TвӮҖ) constrains f via the reproducing structure.

More directly: if f вҲҲ PW_{[вҲ’Оӣ,Оӣ]} has a zero of order m вүҘ 1 at TвӮҖ,
then f(T) = (TвҲ’TвӮҖ)^m В· g(T) where g вҲҲ PW_{[вҲ’Оӣ,Оӣ]} and g(TвӮҖ) вү  0.
This factorization is valid for entire functions of finite exponential
type.  By the Hadamard factorization theorem for entire functions of
finite order (here order 1, type Оӣ), the zeros form a sequence {T_k}
with вҲ‘_k 1/(1+|T_k|ВІ) < вҲһ.  If TвӮҖ is a zero of order m вүҘ 2, then
(TвҲ’TвӮҖ)^{вҲ’m} f(T) = g(T) is entire of the same type Оӣ (since removing
a polynomial factor does not increase the type), and the constraint

    g вҲҲ PW_{[вҲ’Оӣ,Оӣ]},   g(TвӮҖ) вү  0,

together with

    f(T) = (TвҲ’TвӮҖ)^m g(T) вҲҲ PW_{[вҲ’Оӣ,Оӣ]}

is possible only if the Fourier transform satisfies

    fМӮ(Пү) = (вҲӮ_Пү^m [e^{вҲ’iПүTвӮҖ} Дқ(Пү)])|_{appropriate convolution form},

which forces Дқ to compensate for the polynomial (TвҲ’TвӮҖ)^m.  But
(TвҲ’TвӮҖ)^m g(T) вҲҲ PW_{[вҲ’Оӣ,Оӣ]} requires the product of the PaleyвҖ“Wiener
function (TвҲ’TвӮҖ)^m with g to remain in PW_{[вҲ’Оӣ,Оӣ]}, i.e., the product
of an LВІ function with a growing polynomial must stay LВІ.  This forces
g вүЎ 0, contradicting g(TвӮҖ) вү  0.

Therefore f has only simple zeros, unless f вүЎ 0.

Apply to Y: Y вҲҲ PW_{[вҲ’2,2]}, Y is real, Y вүў 0 (since О¶ is not
identically zero on Re(s) = 1/2).  Therefore every zero of Y on в„қ is
simple.

Each zero TвӮҖ of Y corresponds to a zero tвӮҖ = П‘вҳ…вҒ»В№(TвӮҖ) of Z(t), hence
to a zero sвӮҖ = 1/2 + itвӮҖ of О¶.  Simplicity of TвӮҖ as a zero of Y
implies simplicity of tвӮҖ as a zero of Z(t) (since the map T вҶҰ П‘вҳ…вҒ»В№(T)
is a CВ№ diffeomorphism with nonzero derivative at all t вү  t_c, and
the unimodular factor e^{iП‘(t)} in Z(t) = e^{iП‘(t)} О¶(1/2+it) has no
zeros, so the zero orders of Y at TвӮҖ and of О¶ at sвӮҖ are equal).

Every zero of О¶ on the critical line is simple.


PART VI. THE HILBERTвҖ“PГ“LYA OPERATOR AND LOCATION OF ZEROS
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

VI.1  Setup

Work in H вү” PW_{[вҲ’2,2]}, the PaleyвҖ“Wiener space with LВІ inner product

    вҹЁf, gвҹ© вү” вҲ«_в„қ f(T) gМ…(T) dT.

Let рқ’ө вү” {T вҲҲ в„қ : Y(T) = 0} = {П„_k : k вҲҲ в„•} be the zero set of Y on в„қ,
discrete by the entire-function structure of Y.  For each П„ вҲҲ рқ’ө let
K_П„ вҲҲ H be the reproducing kernel at П„:

    f(П„) = вҹЁf, K_П„вҹ©   for all f вҲҲ H.

Since Y(П„) = 0, one has вҹЁY, K_П„вҹ© = 0 for every П„ вҲҲ рқ’ө, so Y lies
in the orthogonal complement of span{K_П„ : П„ вҲҲ рқ’ө}.


VI.2  The multiplication operator H

Define H on the finite linear span of {K_П„ : П„ вҲҲ рқ’ө} by

    H(вҲ‘_П„ c_П„ K_П„) вү” вҲ‘_П„ c_П„ П„ K_П„.

For x = вҲ‘_П„ a_П„ K_П„ and y = вҲ‘_Пғ b_Пғ K_Пғ, compute

    вҹЁHx, yвҹ© = вҲ‘_{П„,Пғ} a_П„ bМ…_Пғ П„ вҹЁK_П„, K_Пғвҹ©,
    вҹЁx, Hyвҹ© = вҲ‘_{П„,Пғ} a_П„ bМ…_Пғ Пғ вҹЁK_П„, K_Пғвҹ©.

The Gram matrix G_{П„Пғ} вү” вҹЁK_П„, K_Пғвҹ© satisfies G_{П„Пғ} = GМ…_{ПғП„}.
