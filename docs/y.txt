THEOREM AND PROOF
=================

Notation and Standing Assumptions
----------------------------------
Let t_0 denote the unique minimum of ϑ(t), approximately 6.29, beyond which
ϑ is strictly increasing. Define σ = ϑ⁻¹ on [ϑ(t_0), ∞). All integrals and
asymptotics are for τ sufficiently large that σ(τ) is well-defined. The
contribution of any fixed initial interval [0, A] to all Cesàro limits below
is O(1/τ_0) and vanishes.


Definitions
-----------
(D1)  V(T) = ∫₀ᵀ |ζ(1/2 + it)|² dt

(D2)  Y(τ) = ζ(1/2 + iσ(τ)) / √(ϑ′(σ(τ)))

(D3)  W(τ) = ∫₀ᵗ |Y(s)|² ds

(D4)  X(τ) = Y(τ) · √(τ / W(τ))


THEOREM
-------
With definitions (D1)--(D4),

    lim_{τ_0 → ∞} (1/τ_0) ∫₀^{τ_0} |X(τ)|² dτ = 1.


PROOF
-----

Step 1.  W(τ) = V(σ(τ)) + C_0   [exact identity]
---------------------------------------------------

W(τ) = ∫₀ᵗ |Y(s)|² ds

     = ∫₀ᵗ |ζ(1/2 + iσ(s))|² / ϑ′(σ(s)) ds.

Substitute t = σ(s). By the inverse function theorem applied to
ϑ(σ(s)) = s, we have σ′(s) = 1/ϑ′(σ(s)), so ds = ϑ′(t) dt.
The factor 1/ϑ′(σ(s)) = 1/ϑ′(t) and the factor ϑ′(t) from ds
cancel exactly. The limits s = 0, s = τ map to t = σ(0), t = σ(τ).
Therefore:

    W(τ) = ∫_{σ(0)}^{σ(τ)} |ζ(1/2 + it)|² dt
         = V(σ(τ)) − V(σ(0)).

Since V(σ(0)) is a finite constant, write

    W(τ) = V(σ(τ)) + C_0,      C_0 = −V(σ(0)), finite.         ···[1]

For all subsequent asymptotics C_0 is absorbed into lower-order terms.


Step 2.  Rewriting the mean square of X  [exact]
--------------------------------------------------

Since W′(τ) = |Y(τ)|² by the fundamental theorem of calculus,

    |X(τ)|² = |Y(τ)|² · τ / W(τ) = W′(τ) · τ / W(τ).

Therefore:

    (1/τ_0) ∫₀^{τ_0} |X(τ)|² dτ
        = (1/τ_0) ∫₀^{τ_0} W′(τ) · τ / W(τ) dτ.              ···[2]


Step 3.  Pointwise asymptotic for W(τ)
----------------------------------------

By Ingham (1926):

    V(T) = T log(T/(2π)) + (2γ − 1)T + E(T),   E(T) = O(T^{1/2}).  ···[3]

The Riemann–Siegel theta function satisfies:

    ϑ(t) = (t/2) log(t/(2π)) − t/2 − π/8 + O(t⁻¹).               ···[4]

Inverting [4]: setting ϑ(σ) = τ gives

    σ(τ) = 2τ / log τ · (1 + O(log log τ / log τ)).                ···[5]

Substituting [5] into [3]:

    σ(τ) · log(σ(τ)/(2π))
        = (2τ / log τ)(1 + O(log log τ / log τ))
          · (log τ − log log τ + O(1))
        = 2τ(1 − log log τ / log τ + O(1/log τ))
          · (1 + O(log log τ / log τ))
        = 2τ + O(τ log log τ / log τ).

The term (2γ − 1)σ(τ) = O(τ / log τ) is absorbed. The error
E(σ(τ)) = O(σ(τ)^{1/2}) = O(√(τ / log τ)) is absorbed. Therefore:

    W(τ) = V(σ(τ)) + C_0 = 2τ + O(τ log log τ / log τ).          ···[6]

This is a pointwise bound, inherited from the pointwise bound on E(T).


Step 4.  Pointwise asymptotic for log W(τ)
--------------------------------------------

From [6], W(τ) = 2τ(1 + ε(τ)) where ε(τ) = O(log log τ / log τ).
Taking logarithms (a continuous operation on pointwise asymptotics):

    log W(τ) = log(2τ) + log(1 + ε(τ))
             = log 2 + log τ + O(log log τ / log τ).               ···[7]

This is pointwise. No differentiation of an O-bound is involved.


Step 5.  Integration by parts
-------------------------------

Rewrite the integrand in [2] using (log W(τ))′ = W′(τ)/W(τ):

    ∫₁^{τ_0} W′(τ) · τ / W(τ) dτ = ∫₁^{τ_0} τ d(log W(τ)).

Integrate by parts:

    ∫₁^{τ_0} τ d(log W(τ))
        = [τ log W(τ)]₁^{τ_0} − ∫₁^{τ_0} log W(τ) dτ.           ···[8]

Evaluate the boundary term using [7]:

    τ_0 log W(τ_0) = τ_0(log 2 + log τ_0 + O(log log τ_0 / log τ_0))
                    = τ_0 log 2 + τ_0 log τ_0
                      + O(τ_0 log log τ_0 / log τ_0).

The boundary contribution at τ = 1 is the constant 1 · log W(1) = O(1).

Evaluate the integral using [7]:

    ∫₁^{τ_0} log W(τ) dτ
        = ∫₁^{τ_0} (log 2 + log τ) dτ
          + ∫₁^{τ_0} O(log log τ / log τ) dτ.

The main term:

    ∫₁^{τ_0} (log 2 + log τ) dτ
        = (τ_0 − 1) log 2 + [τ log τ − τ]₁^{τ_0}
        = τ_0 log 2 + τ_0 log τ_0 − τ_0 + O(1).

The error integral: since log log τ / log τ is positive and decreasing
for large τ,

    ∫₁^{τ_0} (log log τ / log τ) dτ = O(τ_0 log log τ_0 / log τ_0).

Combining via [8]:

    ∫₁^{τ_0} W′(τ) · τ / W(τ) dτ

        = [τ_0 log 2 + τ_0 log τ_0 + O(τ_0 log log τ_0 / log τ_0)]
          − [τ_0 log 2 + τ_0 log τ_0 − τ_0
             + O(τ_0 log log τ_0 / log τ_0)]
          + O(1)

        = τ_0 + O(τ_0 log log τ_0 / log τ_0).


Step 6.  Cesàro limit
-----------------------

The interval [0, 1] contributes O(1) (the integrand W′(τ)·τ/W(τ) has at
worst a removable singularity at τ = 0; see Remark below). Therefore:

    (1/τ_0) ∫₀^{τ_0} |X(τ)|² dτ
        = (1/τ_0)[τ_0 + O(τ_0 log log τ_0 / log τ_0)]
        = 1 + O(log log τ_0 / log τ_0).

Taking the limit:

    lim_{τ_0 → ∞} (1/τ_0) ∫₀^{τ_0} |X(τ)|² dτ = 1.              ∎


Remark on the singularity at τ = 0
------------------------------------
At τ = 0, W(0) = 0, so X(τ) = Y(τ)√(τ/W(τ)) takes the form 0/0.
Since W′(0) = |Y(0)|² > 0 (assuming ζ(1/2 + iσ(0)) ≠ 0), L'Hôpital
gives τ/W(τ) → 1/W′(0) as τ → 0⁺, so X extends continuously with
|X(0)|² = 1. The integrand W′(τ)·τ/W(τ) has a removable singularity
at τ = 0 and is bounded on [0, A] for any finite A. Its contribution
to the Cesàro mean is O(A/τ_0) → 0.


Remark on the role of ϑ near its minimum
------------------------------------------
The function ϑ(t) has a unique minimum at t_0 ≈ 6.29 and is strictly
increasing for t > t_0. The inverse σ = ϑ⁻¹ is therefore well-defined
only for τ > ϑ(t_0) ≈ −3.53. The definitions (D2)--(D4) require σ(τ)
to exist. For the Cesàro limit, the behavior on any bounded initial
interval is irrelevant: one may restrict all definitions to τ ≥ τ_1 for
any fixed τ_1 in the domain of σ, and the contribution of [0, τ_1] to
the Cesàro mean is O(τ_1/τ_0) → 0.
